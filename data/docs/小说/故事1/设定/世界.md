# 0. 标称参数总表（你后文所有计算与场景都可以回扣到这里）

## 0.1 恒星（小一点、但仍可长期适居）

我采用一颗“**老年、相对安静的 M 型红矮星**”作为核心（你可以在设定里明确：它已经度过年轻期耀斑最狂暴的阶段）。

* 质量：(M_* = 0.25,M_\odot)
* 光度：(L_* = 0.008,L_\odot)
* 适居带标称半径（接近地球辐照）：
  [
  a_\mathrm{hab} \approx \sqrt{\frac{L_*}{L_\odot}},\mathrm{AU}
  =\sqrt{0.008},\mathrm{AU}\approx 0.089,\mathrm{AU}
  ]
  数值：(a_\mathrm{hab}\approx 1.33\times 10^{10},\mathrm{m}\approx 1330) 万 km

## 0.2 轨道与“环形外观”的几何

在 (a=0.089,\mathrm{AU}) 的圆轨道上（这是 Layer C 主栖居带所在）：

* 轨道圆周：
  [
  C = 2\pi a \approx 8.37\times 10^{10},\mathrm{m}
  \approx 8366\ \text{万 km}
  ]
* 轨道速度：
  [
  v=\sqrt{\frac{GM_*}{a}}\approx 49.9\ \mathrm{km/s}
  ]
* 轨道周期（“自然年”）：
  [
  T= \frac{2\pi a}{v}\approx 19.4\ \text{天}
  ]

> 这一点对“废弃后误认世界”很重要：**如果居民把季节当自然天象，他们会得到一个很怪的“年”**；你可以让他们用“配光季”当年，用“自然年”当神秘周期（伏笔点）。

## 0.3 辐照度与温度（决定材料与热控）

在适居带 (a=0.089,\mathrm{AU})：

* 辐照度：
  [
  I(a)=\frac{L_*}{4\pi a^2}\approx 1375\ \mathrm{W/m^2}
  ]
  与地球太阳常数（约 1361 W/m²）几乎一致。
* 若取反照率 0.3 的平衡温度：
  [
  T_\mathrm{eq}\approx 255\ \mathrm{K}
  ]

> 这让你可以把“地表生态与气候”写得很像地球——但因为恒星光谱更红，**你可以让“自然阳光”是偏红的**，而“文明想要的白昼”必须由配光网“调白”。

## 0.4 分段（你要的 360 段）

把主栖居带分成 360 段（每段约 1°）：

* 单段弧长：
  [
  \ell = \frac{C}{360} \approx 2.32\times 10^{8},\mathrm{m}
  \approx 23.24\ \text{万 km}
  ]
* 段间安全空窗：500–2000 km（航道、碎片隔离、相位调整走廊）

# 1. 整船架构（五层同心）——把它当“恒星级飞船”的分舱系统写清楚

你想要的效果是：

* **远看像“环形世界”**（一圈发光带）

## Layer A：内侧推进与控光帽（Engine / Cap Zone）

### A1. 结构与位置（“帽子”不是一块，是一片“薄膜群岛海”）

**空间位置建议：**(r=0.015)–0.03 AU（比主栖居带更靠近恒星，能量密度高，利于做遮蔽与光路整流）

* 在 0.02 AU 处，辐照度约 27.2 kW/m²，平衡温度（反照率 0.3）约 538 K。
  这意味着“帽子”材料必须是高反射、耐热、可自修复的薄膜/晶格结构。

**结构单元：**不是“刚体镜”，而是三类单元混编：

1. **反射帆单元（Mirror-sail tiles）**

   * 主要作用：把恒星辐射在宏观上“偏向”某些方向（形成辐射各向异性）
   * 兼任：把可用光导入 Layer B 的配光/输能路由

2. **遮光/屏蔽单元（Shade tiles）**

   * 主要作用：耀斑/高能粒子事件时，为外层提供短时强遮蔽
   * 兼任：让外层的“昼夜/季节”具备硬开关能力

3. **控制与通信骨架（Spine relays）**

   * 稀疏分布的“硬节点”，提供高可靠姿态参考、时钟、束路指令播发
   * 它们是 Layer A 在废弃后最可能仍“以残响形式运转”的部分

### A2. 推进的物理上限（光子动量）与“为什么会很慢”

如果你只靠“让恒星辐射不对称”来推进，推力上限量级是：
[
F \sim \eta\frac{L_*}{c}
]
其中 (\eta) 是“净动量整流效率”（0 到 1 之间；越接近 1，越像把光当火箭喷流）。

对本恒星：

* (\frac{L_*}{c}\approx 1.02\times 10^{16}\ \mathrm{N})
* 恒星质量 (M_*\approx 0.25M_\odot\approx 4.97\times 10^{29}\ \mathrm{kg})
* 加速度上限（(\eta=1)）：
  [
  a_\mathrm{photon}\approx \frac{L_*}{M_*c}\approx 2.05\times 10^{-14}\ \mathrm{m/s^2}
  ]
  这非常小，但它的叙事价值在于：**它永远推、推到文明忘了它在推**。

> 你可以把“完好期的恒星方舟”写成：
>
> * 平时是“巡航态”（(10^{-14}) 量级，几乎不可感知）
> * 关键时期才启用“主动恒星发动机”（下面会给一个合理的工程化表达），把加速度抬到 (10^{-10}\sim 10^{-9}\ \mathrm{m/s^2}) 量级，用于改航向、避让、迁徙窗口。

### A3. statite 的关键约束（无绳索悬停为什么难、为什么废弃后会“漂”）

你前面强调“不要绳索”，那 Layer A 的镜阵要么是：

* **statite（光压平衡引力的准静悬单元）**
* 要么是**轨道单元 + 主动维持（微推力/电推进）**

对“理想反射 statite”单元，平衡条件推得一个非常重要的结论：
它能否悬停**只取决于单位面积质量（面密度）**，与距离无关。

[
\sigma_\mathrm{statite}=\frac{L_*}{2\pi GM_*c}
]

代入本恒星，得到：

* (\sigma_\mathrm{statite}\approx 4.9\times 10^{-5}\ \mathrm{kg/m^2}\approx 0.049\ \mathrm{g/m^2})

这意味着什么？

* 若你希望单元“纯靠光压悬停”，每平方米只能有 **50 毫克级**的质量。
* 这对应“几十纳米量级”的超薄反射膜（还要能抗撕裂、抗微陨石、能自愈）。
* **所以工程上更合理的写法是：**
  Layer A 是“以光压为主的减载 + 以微推力为补偿的准静悬阵列”。
  完好期靠控制系统把它压成一个“帽”；废弃后控制断裂，它就会变成“漂移的薄膜群岛”，遮蔽与反射图案长期偏置。

> 这条物理约束在叙事上非常好用：
> **你不需要写“帽子突然消失”，你只要写它“越来越散、越来越偏、越来越不准”。**
> 居民看到的就是气候带固化、日照走样、神话化的天命分区。

### A4. 完好期功能链路（推进 + 控光 + 防护三合一）

完好期 Layer A 的典型工作流：

1. **常态：光路整流 + 轻度推进（巡航态）**

   * 反射帆单元把一部分恒星光“偏向”船尾方向
   * 剩余部分导入 Layer B 的输能主干（微波/激光束路由）

2. **事件态：耀斑屏蔽**

   * 遮光单元整体转入“遮蔽姿态”，把高能粒子/强光事件的有效剂量压下去
   * 同时 Layer B 降载，Layer C 启动“室内照明模式”（居民会感觉像“阴天几日”）

3. **迁徙窗口：主动发动机耦合（高推力态）**

   * Layer A 的镜阵帽与 Layer B 的高能束路一起工作
   * 以“束路加热/束路推送/磁场导流”驱动恒星风/喷流（你可以用“发动机点火季”来叙事化）

### A5. 废弃后的“自然化表象”（居民会把它当成气候与天象）

当 Layer A 控制失联后，典型可见后果：

* **永久暖带 / 永久寒带**：
  因为遮蔽图案长期固定在某些弧段上，导致那些区域多年偏冷；另一些区域偏热。
  居民会把它解释成“神的赐福与诅咒带”“赤日带/白霜带”。

* **耀斑日成为“天罚日”**：
  以前有屏蔽，后来屏蔽不全，偶尔出现高辐照事件 → 皮肤灼伤、作物异常。
  居民会记成历法里的灾年。

* **光谱异常（强伏笔）**：
  红矮星本光偏红；但文明期的配光会“调白”。
  废弃后调色能力退化，某些日子白昼偏红、偏紫、偏冷白。
  这在神话里会变成“赤日回归”“冷日之冬”。

### A6. 场景切片（完好期 vs 废弃期）

**完好期：舰首维护航行**

> 维护艇穿过 0.02 AU 的光热区，舷窗外不是“天空”，而是无数薄膜单元的边缘闪光——像一片碎银海。每个单元都在微微转角，角度变化小到几乎像静止，但合成起来，恒星光的辉度分布在肉眼里呈现出一种不自然的“缺口”。控制频道里不断报出热负荷、张力、破片修补进度。你知道这不是在修一面镜子，而是在修一只“世界的遮阳伞”。

**废弃期：村落传说里的“天幕裂口”**

> 老人说极北有一道“天幕裂口”，那里太阳会在正午忽然变红，风也会停。没人知道那是遮光单元撕裂后形成的固定光谱孔洞；裂口每年都会缓慢漂移一点点，于是传说也跟着漂移：神在移动，诅咒在迁徙。

## Layer B：戴森云采能与配光网（Power / Light Swarm）

### B1. 结构：多壳层采能 + 束路输能 + 配光/遮光两张网

Layer B 是这艘船最“像文明”的部分：它不是一个建筑，而是一套覆盖内轨道的**电网与光网**。

推荐写法：三层壳 + 两类主干。

* **采能壳（Collector shells）**：0.03–0.07 AU 多层稀疏分布
* **束路主干（Trunk beams）**：固定几条“能量高速路”，把功率从内侧送到外侧
* **配光镜阵（Illumination mirrors）**：靠近 Layer C，每段都有“配光节点”
* **遮光阵（Night-shade array）**：与配光镜阵互锁，负责制造“夜幕”和热控

> 你可以把 Layer B 写成：
>
> * **白天像极细的光纤网**（只有在尘埃/气辉里才看见束路）
> * **夜晚像天幕背后的微光筋络**（一些束路泄漏形成极弱极长的光带）

### B2. 能量账：你要写“世界”，就必须知道它靠什么供能

在适居带，天然辐照约 1375 W/m²。
如果你只靠“自然入射”也能活，但你要的是：

* 人工昼夜（准时开关）
* 人工季节（调强弱、调光谱）
* 工业与推进（高功率）
* 长期稳定（灾害时切换）

所以 Layer B 必须有“远超生态基本需求”的供能冗余。

一个叙事上好用、计算又不夸张的设定是：

* 每个“主弧段”（1°）在完好期拥有可调的输入功率：(10^{14}\sim 10^{15}\ \mathrm{W}) 级

  * (10^{14}\ \mathrm{W}) 足够支撑一个超高工业化的大区（对比：人类当代文明总功率 ~(10^{13}\ \mathrm{W})）
  * (10^{15}\ \mathrm{W}) 则是“重工业+船坞+束路推进”的级别

对应需要多大的采能面积？
在 0.05 AU 处辐照约 4356 W/m²。假设采能与输能总效率 30%（包括转换、传输、接收）：

* 每获取 (10^{14}\ \mathrm{W}) 的可用功率，需要收集：
  [
  A \approx \frac{10^{14}}{0.3\times 4356}\approx 7.65\times 10^{10}\ \mathrm{m^2}
  ]
  即约 7.65 万 km²（相当于一个大国的面积量级，但在太空薄膜里并不离谱）。
* 若是 (10^{15}\ \mathrm{W})，面积乘 10。

> 这给你一个非常清晰的“世界观体量”：
> **每个弧段背后都有一个“看不见的采能领地”。**
> 完好期它是电网；废弃期它是“天上不可触碰的银色神田”。

### B3. 完好期：配光如何制造“像行星一样的昼夜与季节”

你要让居民把它当世界，就要把“日出日落”做得像自然——但你又希望埋伏笔：它其实不是自然。

**最稳妥的工程化写法：**
Layer B 向 Layer C 输出两种光：

1. **散射天光（Sky-light）**：均匀蓝天、云影、柔和亮度
2. **方向性日光（Sun-disk beam）**：一个有边缘、有色温、可变高度角的“太阳盘”

实现手段（你可以混合写）：

* **弧段配光节点**把光束打到每个 Bishop ring 的“日照桅杆/扩散镜”上，扩散成太阳盘；
* **遮光阵**像一圈可编程幕布，把散射天光按时间切断，于是夜幕降临；
* **光谱调制**让白昼色温接近人类习惯（5000–6500K），夜间留下少量偏红的“余辉”，更像真实大气散射。

完好期居民体验会非常像行星：

* 24 小时制
* 有四季（其实是光照强度与高度角的程序）
* 有阴天与风暴（其实是热控与蒸腾的调度）

### B4. 废弃期退化模式：你要的“像坏掉的气候系统一样坏”

Layer B 的优点是“分布式”，所以它很适合你要的废弃世界：

* 它不会一下子死光
* 它会变成**局部仍工作、全局越来越乱**的状态
* 人类会把它当“天象失序”

典型退化路径（非常适合写成历史纪年）：

**阶段 I：失调但可解释（几十年）**

* 日出日落仍大体准时，但季节边界模糊
* 偶发“日光颜色不对”“正午变短”

**阶段 II：错位与重复（百年量级）**

* 某些日子出现“第二个太阳”（束路误指到相邻弧段）
* 某些弧段出现“长夜”（遮光片卡死在关闭姿态）
* 形成最早的大规模宗教解释与禁区划分

**阶段 III：孤岛化（千年量级）**

* 每个弧段/每个大栖居体基本形成自己的“光照制度”
* 有的地方靠残存系统仍是准时文明世界
* 有的地方退化成“以火与农时求生”的中世纪世界
* 交通断裂，知识碎片化，飞船真相变成传说

### B5. 场景切片（完好期 vs 废弃期）

**完好期：配光调度大厅（像航天版电网调度）**

> 墙上不是地图，而是一圈圈轨道与弧段编号。每一段都有一条曲线：散射天光、太阳盘高度角、色温、云层蒸腾预算。调度员说“北三段今晚要做长夜检修”，于是那一段的夜幕会比别处早两个小时降临；居民会收到公告，像收到停电通知一样自然。对他们来说，太阳不是天体，是服务。

**废弃期：牧民记载的“二日之年”**

> 他们的历书里有一年被划了红线：春耕时天空出现两轮太阳，一东一西，光照相互叠加，草场提前枯黄。长者说那是双神争位；你若回头看技术真相，那只是束路分配器在失稳振荡时把日光分裂到了两个接收节点。

## Layer C：主栖居带（“环形世界外观”的本体）

你在这里提出了关键设定：
**每个 1° 分段里有“1 座大型弧段居住体（主居民）”，故事主要发生在这里。**
同时你还希望有许多大型 Bishop ring/空间站作为辅助设施。

最适合的工程写法是：

> **每个 1° 弧段 = 一个“主世界”（主居住体） + 一簇“环城卫星”（大型 Bishop ring 群） + 若干功能站。**
> 远看它们在同一弧线排布，视觉上像一段连续大陆；近看则是高度模块化的太空文明。

### C1. 1° 弧段的“主世界”推荐形态：Arc-Hab（弧段栖居体）

为了实现“居民真的把它当世界”，主居住体最好满足：

* 面积足够大，能承载完整生态与国家级文明
* 天空观感可控（能隐藏太空特征）
* 世界边界可神话化（边墙/禁山）
* 废弃后仍可在局部自洽生存（自治、分区、残存系统）

我给一个非常好写的主居住体模板：**Arc-Hab-Prime（弧段主世界）**
它不是单一 Bishop ring，而是“**一个共享天幕与配光系统的多环聚落链**”，被居民当作“一整个世界带”。

* 由 **6–12 座 BR-3000 级大型 Bishop ring**构成一条链（环与环间距 3000–8000 km）
* 外加若干“线性农业带站”“港口锚点站”“工业站”填充其间
* 最外层共同维护一套“**天空散射层（Sky-veil）**”：高空气溶胶与漂浮薄膜的混合层

  * 它让天空呈现稳定蓝色
  * 抑制真实星空可见度（隐藏 65 分钟自转导致的快速天幕旋转）
  * 也是微陨石防护与辐射屏蔽的一部分
  * 废弃后逐渐破损 → **星星开始“回归”**（超级强的伏笔/揭示机制）

> 这一步很关键：
> 如果你完全开放天空，旋转栖居体会让星空在约 1 小时内转一圈，太容易“露馅”。
> 你要“长期误认世界”，就需要一个能把天空“自然化”的散射层/天幕系统。

### C2. BR-3000 的核心物理量级（用来写“世界边界”“重力差”“禁区”）

取一座典型 BR-3000（半径 3000 km，目标 0.8g）：

* 角速度：
  [
  \omega=\sqrt{\frac{g}{R}}=\sqrt{\frac{0.8\times 9.81}{3\times 10^6}}
  \approx 0.00162\ \mathrm{rad/s}
  ]
* 自转周期：
  [
  T=\frac{2\pi}{\omega}\approx 3870\ \mathrm{s}\approx 64.5\ \text{分钟}
  ]

**可写成世界观细节：**

* 这世界的“自然天象”若不被天幕遮蔽，会出现“天穹飞旋”；
* 因为有天幕，居民只看到“太阳按 24 小时制走”，并把偶尔出现的真实星点当神迹。

**大气质量量级（写资源与灾害）：**
若居住带有效面积按“带面”估算，取宽度 1000 km：
[
A\approx 2\pi R w \approx 2\pi\times 3\times 10^6\times 10^6
\approx 1.88\times 10^{13}\ \mathrm{m^2}
]
1 atm 的大气柱质量约 (P/g\approx 1.03\times 10^4\ \mathrm{kg/m^2})，则：
[
M_\mathrm{atm}\approx 1.9\times 10^{17}\ \mathrm{kg}
]
这是可由彗星/冰小行星供给、也可被“世界管理层”长期维护的量级。
废弃后：漏气、补气失败、分区封闭会成为“末世工程”。

### C3. 弧段站位与交通（完好期“像舰队”，废弃后“像诸国断航”）

完好期：

* 每个 1° 弧段都有若干“锚点站”（非旋转），负责相位保持与交通管制
* 弧段之间的空窗是官方航道，穿越需许可
* 弧段内部靠短程航天器（像海运）在各 BR 间往来
* 你可以把“港口”写成：**中央零重力港 + 下降到旋转世界的电梯**（这也是后期揭示飞船真相的绝佳爆点）

废弃后：

* 锚点站逐步停机，弧段相位开始漂
* 航道变成碎片带、禁区
* 交通断裂造成文化分化：同一条“环形世界外观”上出现多个文明断代层

### C4. 场景切片（完好期 vs 废弃期）

**完好期：从港口落地（重力制度的切换）**

> 飞船停在非旋转的锚点港，脚下是零重力的金属走廊。你沿着“落井”进入转运电梯，警示灯提示：即将进入 0.8g 区域。门开的一瞬间，世界有了方向——风有了重量，人的步伐重新被地面接住。你抬头看到蓝天，太阳盘像被精细打磨过的白金圆，远处两侧的天壁如同永恒山脉。你会忘记自己刚才还漂浮。

**废弃期：农夫的日落（太阳不是落下去，是暗下去）**

> 太阳在黄昏并不滑向地平线，它只是逐渐熄灭，像灯芯收火。云影的移动也像被谁在幕后推动：每一天都几乎相同。孩子问“太阳住在哪里”，大人说住在天壁之后。没人知道太阳其实住在日照桅杆的扩散镜上；而天壁之后，确实有通道——只是那通道通向零重力的骨架与真空。

## Layer D：外层散热与船坞带（Radiator / Yard Belt）

### D1. 为什么散热层在叙事上必须存在（这决定“黑海域”的质感）

只要你有大规模工业与配光网，你就会产生海量废热。废热最终只能靠辐射散出去。

辐射散热通量：
[
q=\sigma_\mathrm{SB}T^4
]
典型温度下：

* 300 K：约 459 W/m²
* 350 K：约 851 W/m²
* 400 K：约 1452 W/m²

这意味着：如果文明总耗散功率达到 (10^{16}\ \mathrm{W})（一万倍人类现今文明），哪怕散热片工作在 350 K，也需要面积：
[
A_\mathrm{rad}\approx \frac{10^{16}}{851}\approx 1.17\times 10^{13}\ \mathrm{m^2}
\approx 1170\ \text{万 km}^2
]
这是一片“大陆级”的散热场——**它在可见光下很暗，但在红外里像海一样发亮**。
所以你写“黑海域”会非常真实：肉眼里它黑，仪器里它亮。

### D2. 完好期：船坞带的功能（它是“飞船的机库”）

* 深空船坞：拼装、拆解、修复大型结构
* 质量驱动器物流：把资源（Layer E）发射成“货流弹道”，在此捕获
* 散热场：吸收整个系统的废热
* 应急隔离：发生碎片灾难时，船坞带可切断与主栖居带的交通，避免级联撞击

### D3. 废弃后：黑海域如何形成（你要的“禁忌之海”）

当控制失效：

* 散热片失去姿态控制，很多会“折叠/翻转/乱飘”
* 船坞遗留大量断裂桁架与未完成结构
* 微小碎片在轨道上形成稠密的危险带
* 这里缺少“配光”，因此肉眼极暗
* 但偶尔会有废热残存导致局部红外强亮（你可以写成“夜里海面有不祥的暗红潮”——其实是热辐射）

### D4. 场景切片（完好期 vs 废弃期）

**完好期：船坞夜班（焊光无声，像雷暴）**

> 船坞没有大气，焊接电弧每一次闪烁都像无声闪电。新一段遮光阵正在装配，薄膜在张紧框架上像水面一样颤动。远处，散热场以肉眼不可见的方式吞吐废热；只有在调试镜头里，它像一片缓慢起伏的暗海。

**废弃期：探险者第一次进入黑海域（“世界”突然没有天空）**

> 穿过天壁后的通道尽头不是山谷，而是一片没有空气的黑。你的火把照不到任何边界，光在真空里显得薄而冷。远处漂着巨大的骨架，表面有直线、有编号、有黄黑相间的警示纹——那种纹路不属于任何自然岩层。你第一次意识到：所谓“天壁”，不是山，是墙；所谓“禁海”，不是海，是机库的坟场。

## Layer E：资源捕获与“墓地轨道”（Resource / Grave Orbits）

### E1. 完好期：资源链是“世界能活下去”的根

你要支撑大气、海洋、土壤、工业，就必须持续补给：

* 冰：水、大气（氧氮也可由冰与矿物化学制取）
* 碳：生态与材料
* 金属：结构、散热、镜阵、船坞

Layer E 的工程形态可以写成：

* 彗星/小行星捕获站（低推力拖曳、或动量交换）
* 分解与精炼站（把资源切成标准货流）
* 货流发射（质量驱动器把货物按弹道射向 Layer D 捕获）
* 墓地轨道（把报废结构送到稳定轨道，避免撞击主栖居带）

### E2. 废弃后：墓地轨道成为“天空里的神”

当文明断航：

* 墓地轨道上的巨大残骸仍会反射恒星光
* 在主栖居体的天空里，它们表现为“移动的星”“缓慢的神舟”
* 偶尔有碎片坠落，成为“天降金属”“发光石”

这能自然地把“飞船世界”的遗迹与宗教传说绑定起来。

### E3. 场景切片（完好期 vs 废弃期）

**完好期：资源拖船与货流弹道（像航运）**

> 拖船把一块冰小行星推上规定轨道，切割臂把它分成标准货块。货块被质量驱动器射出，沿着计算好的弹道飞向外层船坞带。你在控制室看见的是一条条细线——那是货流的航迹，像文明的血管。

**废弃期：天空里的“天舟列阵”**

> 每到冬末，夜空会出现一串极亮的“舟星”，它们沿同一条路径缓慢滑过。祭司说那是祖先归来巡夜。没人知道那其实是墓地轨道上的报废桁架在特定几何下集体反光；它们每 19.4 天重复一次——这是“自然年”的影子。

## 2.1 完好期的总运行链路（读者看见的是“世界稳定”，背后是“系统调度”）

### 链路 1：能量 → 光照 → 气候

1. Layer B 采能壳收集能量
2. 束路把能量送到各弧段配光节点
3. 配光节点决定：

   * 今日太阳盘的轨迹（高度角随时间的曲线）
   * 今日散射天光强度（蓝天亮度）
   * 云量预算（蒸腾与冷凝调度）
   * 夜幕降临与结束时间
4. 各 Bishop ring 的日照桅杆把光“落地”，形成居民眼里的太阳

**居民感知：**日出准时、季节稳定、农业可预测。
**工程真相：**调度中心像电网调度一样每小时更新负荷与云量。

### 链路 2：生态 → 大气 → 补给

1. BR 内部生态循环产生气体交换
2. 大气泄漏由边墙与天幕系统控制在可补偿范围
3. 补给来自 Layer E 的冰与矿物
4. 工业站把补给转化为：水、氮、氧、碳、肥料、结构材料

**居民感知：**世界有雨雪、有河海、有风。
**工程真相：**雨雪是“允许发生的热力过程”，不是天体赠与。

### 链路 3：交通 → 站位 → 安全

1. 弧段锚点站维持相位（避免弧段漂移互相遮蔽/碰撞）
2. 航道空窗保持清洁（碎片清扫）
3. 港口实现“零重力—旋转重力”转换
4. 船坞带承担大修与大型建造

**居民感知：**不同“国度/弧段”之间贸易繁荣。
**工程真相：**这是舰队编队飞行的交通管制系统。

### 链路 4：推进 → 姿态 → 长期迁徙

1. Layer A 形成辐射不对称（慢推）
2. 迁徙窗口启用主动发动机（快推/改航向）
3. Layer B 与 C 进入热控配合（避免过热/过冷）
4. 全系统长期保持“像一艘船一样”的朝向概念（舰首区/舰尾区）

**居民感知：**几乎无感（这是你要的“他们不知道在船上”基础）。
**工程真相：**文明在慢慢搬家。

这是你故事最关键的“可信度骨架”。我给一个能写长篇的时间线，你可按剧情压缩或拉长。

### 阶段 0：断裂当日—断裂十年（系统仍像世界）

* 日照仍准时，但偶尔出现闪烁
* 锚点站还能勉强维持弧段相位
* 交通减少但未断绝
* 这时的人仍知道“我们在太空结构上”（如果你想彻底遗忘，就让这段发生在远古史并被抹除）

### 阶段 I：十年—两百年（“神话化的开始”）

* 配光调度中心失效，日照改为本地自治程序
* 各弧段开始形成不同“日历与季节”
* 局部遮光片卡死 → 诞生“长夜地带”
* 贸易衰落，航道维护失败，碎片增多
* 对后来者而言：这就是“诸神分封世界”的年代

### 阶段 II：两百年—一千年（“国家孤岛化”）

* 锚点站逐渐停机，弧段相位开始漂
* 某些弧段互相遮蔽造成气候极端化（永久冷带/热带）
* 许多 BR 被迫封闭分区（城邦化、生存主义）
* 高技术丢失，仍保留少量“神物”：发光的灯、不会腐蚀的金属门、自动开合的阀

### 阶段 III：一千年以后（“世界自然化完成”）

* 绝大多数人把天壁当山、把港口当神殿、把货流残骸当陨铁
* 仍有极少数地区因为残存供能而保持“近现代”生活（形成强烈对比）
* 星幕系统破损 → 真星空开始出现（你可以把它写成“星辰回归之纪”，作为揭示前奏）

## 3.1 Layer C 地表：同一片平原的“完好期 / 废弃期”

### 场景 C-1（完好期）：农业平原的晨曦——“太阳盘被程序抬升”

平原上没有真正意义的东方与西方（因为地理是弯的），但居民仍然说“东”。日出并不是从地平线“冒出来”，而是天空的亮度先升起，像海水退潮露出浅滩，然后太阳盘在某一刻凝聚成形：一个边缘柔和的圆，色温从偏红缓慢过渡到偏白。露水并不随机，它在固定的两小时窗口内出现，因为蒸腾预算与夜间辐射冷却是被精确控制的；所以老农能用露水判断“今天的云量调度是否正常”。

**可埋伏笔：**

* 日出“先亮后成盘”，不像自然天体。
* 露水窗口规律得过分。
* 云影移动轨迹像扫描线（配光在扫）。

### 场景 C-1（废弃期）：同一片平原的晨曦——“太阳盘偶尔‘缺边’”

几百年后，这里仍有日出。太阳盘依旧出现，但边缘偶尔会缺一口，像被咬过；那缺口会在几分钟内漂移消失。老人说那是“天兽啃日”，孩子会在缺口出现时尖叫。少数学者记下：缺口出现的日子，通常也是“风突然停”的日子——他们不知道这是上游遮光片某个单元的姿态抖动，导致散射天光短时不均匀。

**可埋伏笔：**

* “缺口漂移消失”的动态非常像机械故障，而不像天体遮掩。
* 风与缺口相关（热控系统短时失配）。

## 3.2 天壁（边墙）与“禁山”：完好期是设施，废弃期是神话

### 场景 C-2（完好期）：边墙维护日——“墙上有门、有编号、有涂装”

边墙并非整块岩壁，它是一圈分段的结构体：外层是耐蚀涂层，内层是蜂窝桁架。维护日时，墙体某些位置会亮起蓝色引导灯，提示地表交通绕行。墙脚的检修门打开时，里面不是洞穴，而是笔直的廊道：地面有黄色线，墙上有磨损但仍可识别的符号。工作人员带着磁靴沿廊道向上走，越往里，重力越轻（因为更靠近自转轴），他们的步伐也变得像在水里。

**可埋伏笔：**

* 门的直线与编号。
* 引导灯与绕行公告（世界在“停机检修”）。
* 重力随高度变化（旋转重力特征）。

### 场景 C-2（废弃期）：边墙神殿——“门仍会在某些日子自动开”

废弃后，检修门成了“神口”。它大多数时候紧闭，像岩壁的一道黑缝；但每隔若干天（可能是残存时钟触发），门会发出低沉的振动，缓慢滑开，露出里面无尘的金属走廊。走廊里有些地方仍会亮起微弱的灯，像萤火。祭司说那是“祖先的呼吸”。真正敢进去的人极少，因为进去后会头晕——重力在变轻，人的内耳会失衡，像踏入另一个世界。

**可埋伏笔：**

* 门“按周期自动开”。
* 走廊“无尘”（密闭过滤残响）。
* 进入后重力变轻（强烈提示“不是山洞”）。

## 3.3 轴向“天空散射层（Sky-veil）”：完好期隐藏真相，废弃期让星空回归

### 场景 C-3（完好期）：夜晚看不到星——“夜空是一种服务”

夜里，天空是深蓝而不是黑，像浓墨里兑了水。偶尔能看到很淡的星点，但星点位置固定，像镶嵌在天幕里，不随时间转动。孩子会背诵这些星点的名字，像背诵图案。成年人才知道那是“夜空投影”：真正的星空会暴露结构自转与外部设施的存在，所以系统选择给居民一个“心理更舒适的夜”。

**可埋伏笔：**

* 星点不转动，像贴图。
* 夜空亮度太稳定。
* 星座“像印刷品”。

### 场景 C-3（废弃期）：星星第一次出现——“天幕破了”

某一年冬末，天幕的散射层开始稀薄。夜里，蓝不再均匀，出现一片片更深的黑。起初只是几颗星，亮得刺眼；然后星越来越多，甚至出现一条淡淡的银河。最可怕的是：这些星会“走”，而且走得很快，像巨轮旋转。民众恐慌，认为天空在塌陷。你作为作者可以在这里做第一次重大揭示：**天空在以约一小时一圈的速度旋转**——这是旋转栖居体的惯性真相开始渗出。

**可埋伏笔：**

* 星空快速旋转（读者会立刻意识到“有问题”）。
* 民众把它当“天塌”，符合误认世界的心理。
* 这是你可以从“神话世界”转向“硬科幻揭示”的最佳桥段。

## 3.4 Layer B 束路事故：二日、长夜、冷日（你要的“坏气候像坏机器”）

### 场景 B-1（完好期）：配光检修公告——“太阳也会停机维护”

广播说：今晚北二段将进入“低天光模式”，日落提前两小时，明晨日出延后半小时。城市照明将补偿。居民听完只是点头：他们会囤一点灯油或充电，像现代人应对停电。天空按公告变化：夜幕确实早到，星点投影更亮了一点，像“服务加班”。

**可埋伏笔：**

* 日落日出可公告、可提前延后（强烈非自然）。
* 夜幕“按计划”降临。

### 场景 B-1（废弃期）：二日之灾——“束路振荡把太阳复制”

正午，太阳盘本应在最高点，但它突然分裂成两个：一个偏东，一个偏西。光照叠加后热浪扑面而来，河面起雾，鸟群混乱。傍晚时，其中一个太阳忽然熄灭，另一个还在，像神明争斗后胜者独存。第二天，太阳恢复正常，仿佛昨日只是幻象。你可以在后记或后期揭示：那是束路分配器振荡，两个接收节点同时被点亮。

**可埋伏笔：**

* 太阳“出现/熄灭”的方式像开关。
* 热浪与生态异常是工程事故的自然后果。
* “只发生一天”很像系统抖动而非天体改变。

### 场景 B-2（废弃期）：长夜带——“遮光片卡死的世界”

某些地区进入“长夜”。不是极夜那种逐渐变暗，而是某一天黄昏后太阳再也没有回来。散射天光也不再升起，只有很弱的红色余辉像煤火。人们迁徙，建立“火城”，用热与光维持作物。长夜带与正常带之间形成边境：边境上常年大风，因为热差巨大。传说说那是“夜之门开了”，而你知道那是遮光阵卡死在关闭姿态。

**可埋伏笔：**

* 长夜是“突变式发生”，符合机械故障。
* 边境大风是热控失配造成的物理现象，真实可感。

## 3.5 Layer D 黑海域：完好期是机库，废弃期是“外侧世界”

### 场景 D-1（完好期）：船坞观礼——“新环出厂”

新一座 BR 的桁架在船坞里缓慢旋转，像一枚正在成形的行星环。工人用拖船把散热翼一片片挂上去，像给巨兽装鳞甲。剪彩时没有欢呼，因为真空里声音传不开；只有无线电频道里的短促掌声。远处的散热场像黑色幕布，吞掉所有废热，把文明的喧嚣变成无声的红外波。

**可埋伏笔：**

* “剪彩无声”强调真空。
* 新环像“出生”，强化文明尺度。

### 场景 D-1（废弃期）：探险者越墙——“第一次看到真空”

当他们穿过边墙通道，最后一道门打开，风突然消失。火把的火焰被抽干，只剩一缕烟立刻散开。面前是一片黑：没有空气、没有回声。远处漂着巨大的结构骨骼，上面有直线、有齿轮、有镜面残片反光。你可以让主角在这一刻第一次意识到：**所谓“世界的外侧”不是另一片大陆，而是宇宙。**

**可埋伏笔：**

* 火焰熄灭（真空）是最直观的物理揭示。
* 直线与编号击穿“自然解释”。

# 4. 埋伏笔的方法（系统化工具箱：每一条都能在前期像民俗，后期像证据）

你要的“后面发现是飞船才更有冲击力”，核心不是“隐藏所有线索”，而是：

* **线索一直都在**
* 但前期都能被角色用“自然/宗教/民俗”解释
* 后期一旦拿到关键证据，这些线索会整体翻转成“工程证据链”

下面给你一套“伏笔工具箱”，你可以按章节撒点，不会显得刻意。

## 4.1 天象类伏笔（最隐蔽、最好用）

1. **日落不是落下，是变暗**
2. **太阳盘偶尔缺边/闪烁**（遮光/配光抖动）
3. **某些年份出现二日**（束路误指）
4. **“自然年”只有 19.4 天**（如果有人坚持用星象记年，会得到怪结果）
5. **星空不转 / 旋转太快**（取决于天幕状态）
6. **星座“像印刷品”**（投影星空）

## 4.2 地理类伏笔（角色会当成“神话地貌”）

1. **天壁山脉过于均匀、过于直**（边墙分段）
2. **河流的曲率不合直觉**（大尺度曲面）
3. **某些“峡谷”内部永远无尘**（通风过滤）
4. **山体内有“黄线”“符号”“门”**（维护通道）

## 4.3 物理规律类伏笔（最硬核，但要少量使用）

1. **重力随高度略变轻**（旋转重力：(g(h)\approx g_0(1-h/R))）
2. **抛物线偏转异常明显**（强科里奥利效应）
3. **风暴旋向几乎永远一致且尺度偏小**（快速自转）

> 这些不要一次讲透，只要写成“本地人熟悉的怪规律”，后期再被角色用物理解释。

## 4.4 遗物类伏笔（最适合“探险章节”）

1. **不会腐蚀的金属门**
2. **重复出现的编号（段号、泊位号）**
3. **黄黑警示纹**
4. **无法理解的材料（薄膜碎片、镜面残片）**
5. **偶尔启动的灯与阀**（“自动残响”）

你说“故事主要发生在 1 座大型弧段主居民上”。为了让你写起来像写一颗行星，我建议你把这个主舞台弧段明确成“五个可反复出现的地理区域”，每个区域都能挂一个伏笔主题。

# 6. 一个关键提醒（保证“误认世界”长期成立的必要条件）

你想要“大家不知道这是飞船，把它当作正常世界”。在硬科幻自洽上，最容易露馅的点有两个：

1. **星空与天幕**：如果完全开放天空，旋转栖居体的星空旋转会非常快（约 1 小时一圈）。

   * 解决：必须有“天空散射层/天幕投影/云顶系统”把真实星空长期压住；
   * 废弃后逐步破损，星空回归，成为揭示前奏。

2. **“年”太短**：自然轨道周期 19.4 天。

   * 解决：文明期的“季节”完全由配光程序定义；居民的历法与农时基于配光季（比如 360 天）；
   * 少数观星者会发现“天舟列阵每 19.4 天复现一次”，作为神秘周期伏笔。

这两个点一旦处理好，你的“误认世界”会非常稳，且揭示时冲击巨大。
