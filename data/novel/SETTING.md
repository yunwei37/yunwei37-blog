# 《诅咒魔女的千人枕》（暂定名）

**作者**：尘简

## 核心概念
**双重生 + 变身 + 欢喜冤家 + 修仙复仇 + 创意诅咒战斗**

> "前世你杀了我一次，这辈子我要睡你一千次。" —— 不是威胁，是承诺。

## 世界观

### 大陆设定
**玄天大陆** —— 修仙世界，强者为尊

- **正魔两道**：表面对立，实则利益交织
- **宗门林立**：大宗门把持资源，小宗门艰难求生
- **散修艰难**：没有靠山的修士，活得像老鼠

### 核心冲突
- 前世：墨尘（魔道药圣）被霜华仙子（正道圣女）斩杀
- 今生：两人都重生了，身份逆转
- 墨尘 → 苏棠（落魄宗门废柴弟子）
- 霜华仙子 → 柳霜寒（大宗门天才弟子）

### 修炼体系（重点：创意战斗）

#### 诅咒道（主角专属）
被视为邪道中的邪道，正道不屑，魔道也看不起。但在主角手中变成最阴间的存在：

**诅咒种类：**
1. **痛觉系** - 痛觉倍增、痛觉链接、幻痛术
2. **反伤系** - 荆棘之拥、诅咒反弹、伤害存储
3. **精神系** - 噩梦侵蚀、强迫诅咒、记忆混淆
4. **厄运系** - 霉运缠身、关键时刻掉链子、失误连锁
5. **毒煞系** - 延迟毒素、情绪毒素、因果毒素

**主角独创：诅咒符箓**
将诅咒封印进符箓，可以交易、远程激活、组合使用：
- 【痒痒符】：贴谁谁痒三天，越挠越痒
- 【霉运符】：三天内运气降低50%
- 【分手符】：恋情的双方会突然互相看不顺眼
- 【社死符】：在最关键的时刻说出最丢人的话

（诅咒越"小"，越难解除——羞辱程度与强度成反比）

#### 冰心诀（女主二）
正道第一功法，专修冰系神通：
- 霜天意境、冰封千里、天霜剑诀
- 但修炼者必须断情绝欲——断得越干净，威力越强
- 柳霜寒前世就是因为对墨尘动了心，才被宗门清洗

### 势力分布

**正道**
- **玄冰宗**：正道第一宗门，柳霜寒所在
- **天剑门**：剑修圣地
- **丹霞谷**：炼丹大宗

**魔道**
- **万毒门**：墨尘前世所在（已覆灭）
- **血河宗**：血修大宗
- **影杀殿**：刺客组织

**第三方**
- **天云商会**：大陆第一商会
- **散修联盟**：被压迫者的聚集地
- **天道盟**：正道联盟，以"斩妖除魔"为名猎杀诅咒道等"邪道"修士

## 人物设定

### 女主一：苏棠 / 前世：魔道药圣·墨尘

**外貌**
- 金色长发（前世黑发），天蓝色眼睛
- 身材娇小（155cm），平胸（本人很在意）
- 外表柔弱无害，像一朵小白花

**性格**
- 屑系主角：爱钱、会骗人、嘴甜心黑
- 有底线：不害无辜，但对敌人绝不手软
- 爱吃醋、占有欲强、但嘴硬不承认
- 内心戏丰富，吐槽役

**前世身份**
- 魔道药圣，毒术通神，炼丹无双
- 被正道追杀，最终死于霜华仙子之手
- 死前诅咒："若有来世，必让你求生不得求死不能"

**今生身份**
- 落魄宗门青阳宗的"废柴"弟子
- 被师门放弃，扔到外门自生自灭
- 诅咒道天赋（被视为废物道法）

**战斗风格**
- 绝不硬刚，能用阴招绝不用正招
- 战前布陷阱、战中用诅咒、战后追债
- 名言："打架为什么要站着打？躺着赢不行吗？"

**金手指**
- 前世炼丹知识 → 创造独特诅咒丹药
- 可以把诅咒封印进符箓
- 对毒药和诅咒的绝对免疫

---

### 女主二：柳霜寒 / 前世：霜华仙子

**外貌**
- 黑色长发，冷艳高贵
- 身材高挑（170cm），完美曲线
- 冰山美人外表，实则内敛闷骚

**性格**
- 表面高冷禁欲，实际内心戏很多
- 傲娇典范：嘴上说讨厌，行动上很诚实
- 有严重的责任感，对自己要求极高
- 前世动了心，却亲手杀了自己动心的人

**前世身份**
- 正道第一宗门玄冰宗的圣女
- 奉命追杀墨尘，亲手斩杀
- 后来被宗门清洗——因为她动了心

**今生身份**
- 大陆第一商会林家的独生女
- 同时也是玄冰宗的天才弟子（双重身份）
- 被所有人仰望，内心却很孤独

**战斗风格**
- 正统冰系神通，实力强劲
- 习惯正面对决，不屑阴招
- 但在苏棠影响下，逐渐"灵活"

## 感情线发展

### Phase 1: 仇人重逢（1-50章）
- 两人认出对方前世身份
- 苏棠开始"复仇计划"——让柳霜寒爱上自己
- 各种阴招阳谋，互相试探
- 柳霜寒逐渐发现——苏棠就是墨尘

### Phase 2: 欢喜冤家（51-150章）
- 被迫合作对抗共同敌人
- 日常互坑，但关键时刻互保
- 暧昧升级：同床共枕、互相吃醋
- 苏棠嘴硬："我才不是为了你！"

### Phase 3: 相爱相杀（151-250章）
- 感情升温，苏棠终于承认喜欢
- 柳霜寒的内心挣扎：正道vs感情
- 第三者介入，引发吃醋狂潮
- 关键战役中互相牺牲保护对方

### Phase 4: 并肩作战（251-350章）
- 联手对抗玄冰宗
- 揭开前世真相
- 柳霜寒与宗门决裂
- 两人正式在一起

### Phase 5: 复仇与救赎（351-400章）
- 面对大陆级危机
- 诅咒道被认可
- 两人联手拯救世界
- HE结局

## 剧情大纲

### 第一卷：废柴弟子的逆袭（1-50章）
- 重生醒来，发现自己变成了女的
- 加入青阳宗外门，被看不起
- 用诅咒道打脸，震惊全场
- 偶遇柳霜寒，认出对方是霜华仙子
- 制定"复仇计划"——让她爱上自己

### 第二卷：诅咒符箓的诞生（51-100章）
- 开发符箓系统，开始做生意赚钱
- 遭遇散修危机，被迫与柳霜寒合作
- 两人并肩战斗，暧昧升级
- 天道盟出现，猎杀诅咒道修士

### 第三卷：商会风云（101-150章）
- 被卷入商会权力斗争
- 柳霜寒被逼联姻，苏棠大闹宴会
- 公开身份：柳霜寒是她的
- 玄冰宗介入，危机升级

### 第四卷：正魔之战（151-200章）
- 正魔两道全面开战
- 苏棠被迫选边站
- 柳霜寒内心挣扎
- 两人确认感情

### 第五卷：前世真相（201-300章）
- 揭开前世恩怨的真相
- 柳霜寒与玄冰宗决裂
- 联手对抗宗门
- 甜蜜日常穿插

### 第六卷：诅咒女王的崛起（301-350章）
- 建立诅咒道传承
- 对抗天道盟
- 大陆危机
- 最终之战

### 第七卷：余生（351-400章）
- 战后重建
- 诅咒道被认可
- 两人正式成婚（道侣）
- 甜蜜日常收尾

## 写作风格

1. **轻松幽默**：大量吐槽、玩梗、日常搞笑
2. **战斗创意**：每场战斗都要有新花样
3. **感情张力**：推拉、暧昧、吃醋、嘴硬
4. **高潮爽点**：打脸、反转、以弱胜强
5. **日常甜度**：同居、互宠、小确幸

## 章节模板

每章约8000-10000字，结构：
- 开头：承接上文/日常互动
- 中间：剧情推进/战斗/感情
- 结尾：钩子/悬念/搞笑收尾
