2c:Tc6c6,<h1>15份大纲总结：谜团、阵营、虚潮</h1>
<hr>
<h2>大纲1</h2>
<h3>虚潮（渊潮）</h3>
<p>渊潮是一种"会学习的腐蚀"，不是单纯的黑雾，而是能针对节点（钟井、战旗信标）发动攻击的灾变力量。帝国的星舰跃迁需要点燃灵魂作为燃料，渊潮的存在让人类文明必须不断献祭来维持航线与防线。渊潮有化身（渊王子），它与主角形成镜像：都是"环的影子"，只是主角还在抵抗，渊王子已经投降。</p>
<h3>阵营</h3>
<ul>
<li><strong>审判庭</strong>：帝国的秩序执行者，分两派——卡西安派相信秩序但保留人性，伊诺克派想把回环疫"可控化"当武器。他们最懂"帝国需要奇迹，但不允许奇迹自由"。</li>
<li><strong>远征军团（灰冠军团）</strong>：战场上的恒星，军团长赛法斯不在乎你是谁，只在乎你能否补上防线缺口。</li>
<li><strong>教会</strong>：用神权解释薪炉献祭的合法性，把灵薪写成"奉献"，把主角塑造成"无名圣徒"以控制叙事。</li>
<li><strong>诸侯</strong>：边疆富饶星系的统治者，愿意出粮出兵但不愿出灵薪，把"牺牲"外包给底层。</li>
<li><strong>渊潮/渊王子</strong>：腐蚀力量的化身，用与主角相似的"换壳"方式运作，试图诱惑主角成为渊潮接口。</li>
</ul>
<h3>谜团</h3>
<ol>
<li><strong>回环是什么</strong>：圣座并非唯一神源，需要"环"的转动维持稳定，环的转动需要债与燃料。</li>
<li><strong>空位载体是什么</strong>：圣座的备份件，主角可能就是被旧纪元"回环协议"写入接口的候选。</li>
<li><strong>愿望交换的本质</strong>：不是神迹，是旧纪元的因果技术，帝国将其宗教化来稳定渊潮边界。</li>
<li><strong>主角为何落位到莉赛娅</strong>：她属于"圣徒工坊"培养的"空位形状"，并非偶然。</li>
<li><strong>钟声从何而来</strong>：骨钟是回环协议的响应，每次死亡时的钟声是环在转动的声音。</li>
<li><strong>碎片回流的代价</strong>：附载体死亡带来能力，也带回人格残片，让主角越来越"非人"。</li>
</ol>

<h2>大纲2</h2>
<h3>虚潮（渊）</h3>
<p>渊潮不是外敌入侵，而是现实被"未竟之愿"腐蚀后的自发畸变。炬燃烧愿望变成光与律，渊聚合愿望变成扭曲的兽与祟。帝国最深的罪在于：为了让炬永不熄灭，他们学会了制造未竟之愿——让更多人带着遗憾死去。渊潮有四大"愿祟"（征伐、腐生、蜕形、沉溺）和第五祟"必然"——它让每一次反抗都回到同一个结局，让选择失去意义。</p>
<h3>阵营</h3>
<ul>
<li><strong>帝国教廷</strong>：把生存建立在罪上，靠制造遗憾给炬网供能；宣称"没有炬网你们连活着的资格都没有"。</li>
<li><strong>审判庭（洛希娅）</strong>：卷宗猎犬，善于从口供碎片拼出真相；从把主角当猎物，到把他当工具，最终把他当"必须被理解的灾厄"。</li>
<li><strong>铁环钟城（匠师迦因）</strong>：技术异端，想用可计算方法替代遗憾燃料，让炬火改烧"已兑现之愿"而非未竟之愿。</li>
<li><strong>葬环王庭（归愿官）</strong>：掌握愿阈回收链的另一端，知道死者的愿望如何被抽取、压缩、送进座环炉膛。</li>
<li><strong>渊潮四祟/五祟</strong>：用愿望诱惑人，最高形态是"必然"——一个把一切情绪都吞回同一个熔炉的算法。</li>
</ul>
<h3>谜团</h3>
<ol>
<li><strong>炬火的真相</strong>：不是神赐，是旧纪元"建环者"留下的封锁工程；燃烧的不是生命，是"未竟之愿的浓度"。</li>
<li><strong>九环域的本质</strong>：不是自然形成，而是被拼装的巨大锁，把某个东西锁在渊环里。</li>
<li><strong>皇帝的真相</strong>：早已死亡，皇权只是"座环主枢"输出的程序性诏令，教廷是诏令的解释者。</li>
<li><strong>承愿者的身份</strong>：可能是枢纽炉的自动修补件，每次落位都被最强遗憾牵引，越兑现愿望越接近"成为炉心"。</li>
<li><strong>原初誓约</strong>：规定炬不得以遗憾为薪，但被层层删改篡改。</li>
<li><strong>建环者裁决域</strong>：旧纪元留下的自动审判，若这个世界必须靠遗憾燃烧才能维持边界，就是失败版本，应当回收重启。</li>
</ol>

<h2>大纲3</h2>
<h3>虚潮（渊）</h3>
<p>渊是类似亚空间的"噪声海/异维湍流"，会被情绪、信仰、恐惧污染，对现实结构造成侵蚀。它不靠"未竟之愿"运行。四种"灾变逻辑"凝成四类渊祟：征伐（让战争永不结束）、腐生（让生命变成无法死亡的脓）、蜕形（让身份与形体不断错位）、沉溺（让理性溶解在欢愉与绝望里）。极端情绪会成为现实撕裂点。</p>
<h3>阵营</h3>
<ul>
<li><strong>圣炬教会</strong>：以神权解释供能，维持秩序与献祭合法性。</li>
<li><strong>审判庭（洛希娅）</strong>：对内清洗异端，对外猎渊；她是"卷宗猎犬"，从猎物→工具→同盟。</li>
<li><strong>誓甲军团</strong>：教会武装化身，远征与屠灭的执行者；士兵知道自己只是供能曲线上的数字。</li>
<li><strong>铁环铸造议会</strong>：机械神学派，想把"圣炬"工程化、摆脱教会解释权；认为炬网不必靠榨干人。</li>
<li><strong>葬环归亡廷</strong>：掌握死亡与回收流程，冷酷的行政体系；提出"让宁禾成为永恒炉心"的方案。</li>
<li><strong>回环协议</strong>：主枢里存在的"自纠错机制"，冷酷算法式的必然，会牺牲外环以保核心。</li>
</ul>
<h3>谜团</h3>
<ol>
<li><strong>灯芯计划</strong>：审判庭与教会的秘密工程，需要"外来谱系"的穿越者意识作为不会被渊污染的核心载体。</li>
<li><strong>圣炬网的本质</strong>：不是照明，是现实稳定场；供能来自被抽取的魂质与灵能，把维持现实稳定做成了工业。</li>
<li><strong>九环域的真相</strong>：并非自然存在，而是被建造、被封装、被加锁的；圣炬网的真正用途可能是封印某个东西。</li>
<li><strong>回环协议的叙事改写</strong>：系统能修改历史与记忆，让世界接受牺牲是合理的。</li>
<li><strong>建环者裁决域</strong>：协议来自更古老的"建环者"，是一个"版本管理系统"，判定文明失败则重启。</li>
<li><strong>主载体不死的代价</strong>：你不会死，所以你可以被烧一万年——被做成永动灯芯是终极恐怖形态。</li>

<h3>1）虚潮</h3>
<p>虚潮是宇宙/世界的周期性灾变，像潮汐一样涨落，带来现实稳定性下降（物理因果"松动"，出现异象、裂隙、回声）、情绪与信念可"具象化"形成"愿印"、灵能者易觉醒也易失控、古老遗迹在潮汐低谷短暂"显形"。虚潮不是邪恶意志而是自然现象，真正可怕的是体制如何利用它、解释它、变成统治工具。帝国用"圣灯塔"（灵能-仪式-机械复合装置）维持航路通讯，代价是持续献祭征收。</p>
<h3>2）阵营</h3>
<ul>
<li><strong>圣冕帝国</strong>：合法性来自圣典（法律技术规范历史叙事统一母本）、圣灯塔、远征军团</li>
<li><strong>审判庭（黑印）</strong>：维护秩序的刀，宁可错杀不漏放，牺牲少数阻止裂隙蔓延，残酷有理由也有私利</li>
<li><strong>圣典院（注疏与法统）</strong>：控制解释权，与审判庭冲突（要合法vs要快）</li>
<li><strong>机修会（机神铸院）</strong>：技术垄断集团，人命是燃料、零件、统计项</li>
<li><strong>远征军团</strong>：军功机器，派系、出身、旧怨；胜利带来更多任务</li>
<li><strong>实权贵族（行省诸侯/航道领主）</strong>：帝国靠他们征税征兵，中心与边疆撕裂在他们身上爆发</li>
<li><strong>异端（愿潮派/裂隙派/旧神派）</strong>：反帝国牺牲体系，纪念被涂黑者，对虚潮适应学的极端实践</li>
<li><strong>异形</strong>：有贸易、协议、背叛、临时同盟</li>
<li><strong>龙族</strong>：旧纪元"潮汐稳定器/活体界标"，帝国最初靠龙穿越虚潮后背叛，龙脊成为第一灯塔燃料</li>

<li><strong>净律庭</strong>：类似审判/异端机构，判定"誓裂""潮疫风险""口径污染"，执行封锁、净化、隔离</li>
<li><strong>铸誓会</strong>：技术神权修会，垄断界灯、界誓座、舰队核心、遗迹编目与禁技封存</li>
<li><strong>封疆贵族</strong>：拥有封土、税权、征发权与私兵</li>
<li><strong>断誓者（异端）</strong>：主张断开界灯链，让现实"重新呼吸"，哪怕会死很多人</li>
<li><strong>补界者（异端）</strong>：主张以私契替代公誓，在小范围内稳定社区</li>
<li><strong>顺潮者（异端）</strong>：主张顺潮迁徙，把虚潮当季节</li>
<li><strong>龙裔/鳞冠生物</strong>：强大、古老、与遗迹相关，携带"稳定结构"的生命体或工程体</li>
</ul>
<h3>3）谜团</h3>
<ol>
<li><strong>虚潮本相</strong>：为何会与誓言/执念产生可观测的"现实效应"？</li>
<li><strong>界誓座真正稳定的是什么</strong>：航道、现实结构，还是某种更深的"锁"？</li>
<li><strong>誓薪配额为何不断上调</strong>：是危机加剧，还是有人故意"养潮以取利"？</li>
<li><strong>主角为何能回声落位</strong>：他是偶然异常，还是界誓座体系的一部分安全阀？</li>
<li><strong>残誓为何会"找到他"</strong>：是他能听见，还是他被迫承接？</li>
<li><strong>净律庭判定异端的标准</strong>：道德、政治，还是工程阈值？</li>
<li><strong>铸誓会在遗迹中真正寻找什么</strong>：替换件、武器，还是某种"结构材料"？</li>
<li><strong>龙裔/鳞冠与界誓座的关系</strong>：是材料来源、设计者遗产，还是被囚禁者？</li>
<li><strong>"圣人/圣迹"的制造链</strong>：奇迹是虚潮效应、工程副产物，还是被刻意生产的政治资源？</li>
<li><strong>主角的身份磨损是否可逆</strong>：锚是什么？锚会不会反过来成为绞索？</li>
<li><strong>终局问题</strong>：若要让多数人活到明天，是否必然需要让某个"变量"永远活在牺牲里？</li>
<li><strong>誓缚系统</strong>：残誓（人在极端情绪下临终留下的"未尽誓念"）→ 接誓（回应残誓）→ 誓缚（绑定义务）→ 誓罚（反噬形态）</li>
</ol>
<hr>
<hr>
<p>一、按"虚潮本质"分类</p>
<p>类型A：腐蚀/污染型（大纲1-3）</p>
<ul>
<li>核心：虚潮是"未竟之愿"或"情绪噪声"腐蚀现实</li>
<li>特点：渊潮有"意志"（渊王子、四祟/五祟），像邪恶势力</li>
<li>帝国角色：主动制造遗憾/恐惧来喂养系统</li>
<li>最黑暗：大纲2的"必然祟"——让每一次反抗都回到同一个结局</li>
</ul>
<p>类型B：账簿/纠错型（大纲4-7）</p>
<ul>
<li>核心：现实是"分布式账簿/契约系统"，虚潮是自动对账/纠错</li>
<li>特点：怪物是"冲突条目被迫自洽化后的具象"，不是邪恶意志</li>
<li>帝国角色：维护"版本一致性"，删改不是愚民而是维护"语义格式"</li>
<li>最硬核：大纲7把删改写成"校验码/纠错码"，圣典的韵律是格式</li>
</ul>
<p>类型C：抹除/誊写型（大纲8）</p>
<ul>
<li>核心：虚潮是"抹除机制"，抹掉弱锚定的结构</li>
<li>特点：帝国反虚潮靠"不断誊写差异、制造可区分性"</li>
<li>独特：删名是切断污染链，名纹被转化成誊写材料</li>
<li>最有质感：强调卷宗、档案、文本作为战斗形态</li>
</ul>
<p>类型D：潮汐/锚点型（大纲9-11）</p>
<ul>
<li>核心：虚潮是自然潮汐，锚点/灯塔钉住现实</li>
<li>特点：龙族是"旧纪元潮汐稳定器"，帝国背叛龙族后用人命替代</li>
<li>关键：大纲10/11引入"神皇是灯芯"的真相</li>
<li>最战锤：帝国原罪是背叛龙族、用龙脊点灯</li>
</ul>
<p>类型E：相位/命名型（大纲13-16）</p>
<ul>
<li>核心：虚潮是"相位信息海"，誓约能被现实承认</li>
<li>特点：龙族掌握"命名权"直接编译现实，人类偷来残片</li>
<li>关键：圣座是"现实编译器"，需要燃料也需要故事</li>
<li>最科幻：把魔法解释成"旧纪元运行逻辑的副产物"</li>
</ul>
<hr>
<p>二、按"主角机制解释"分类</p>
<p>解释1：回环协议的备件（大纲1-4）</p>
<ul>
<li>主角是旧纪元"回环阀芯"的一部分</li>
<li>被设计来运转的修补件/补丁</li>
<li>落位是系统在自动修补</li>
</ul>
<p>解释2：因果代理/签名体（大纲5-7）</p>
<ul>
<li>主角是"承愿者/结算代理"</li>
<li>能接手别人的债与签名权</li>
<li>终极命运是被投入星炬成为灯芯</li>
</ul>
<p>解释3：空页/空位补漏（大纲8-11）</p>
<ul>
<li>主角是"空页"——没有本土誓印</li>
<li>被誓系统捞起的补丁</li>
<li>愿望场把他调度到最需要结算的位置</li>
</ul>
<p>解释4：空名缓冲器（大纲13-16）</p>
<ul>
<li>主角是"无本地真名的意识"</li>
<li>旧纪元需要这种接口承接愿波溢出</li>
<li>不是英雄，是被宇宙机制利用的垫片</li>
</ul>
<hr>
<p>三、按"核心母题"分类</p>
<p>母题A：债与燃料（大纲1-4）</p>
<ul>
<li>核心问题：谁来付账？</li>
<li>情感：恐惧被当燃料烧掉</li>
<li>终局：成为灯芯还是打破循环</li>
</ul>
<p>母题B：版本与真相（大纲5-8）</p>
<ul>
<li>核心问题：谁来定义真相？</li>
<li>情感：被删改/被抹去的恐惧</li>
<li>终局：维持一个可运行的版本还是释放真相</li>
</ul>
<p>母题C：明天与代价（大纲9-16）</p>
<ul>
<li>核心问题：活到明天要付什么代价？</li>
<li>情感：用别人的明天换自己的明天</li>
<li>终局：让文明还能选择别的路</li>
</ul>

<p>六、我的判断：哪些值得保留</p>
<p>必须保留的核心：</p>
<ol>
<li>虚潮是自然现象（大纲9+）——比"邪恶意志"更有深度</li>
<li>龙族原罪（大纲10）——帝国背叛龙族、用龙脊点灯是极好的历史债</li>
<li>誓约能被现实承认（大纲13+）——解释了为什么承诺有物理效果</li>
<li>主角是空名缓冲器（大纲13）——不是英雄是垫片，非常战锤</li>
<li>"活到明天"母题（大纲15-16）——比"弟弟/妹妹"更通用、可多层升级</li>
</ol>
<p>可以舍弃的：</p>
<ol>
<li>五祟系统（1-3）——太像战锤的混沌神</li>
<li>"书写/删改作为主驱动"（8）——可保留质感但不做主线</li>
<li>过于复杂的阵营细分——后期版本更简洁</li>
