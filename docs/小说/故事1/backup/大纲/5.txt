2c:Tc5ec,<h1>《愿潮圣座》叙事风格大纲（约300万字工程版）</h1>
<blockquote>
<p>题眼一句话：<strong>一个“会死但死不掉”的普通穿越者，在神权帝国的远征与审判之间反复换壳、替愿而活；他每赢下一场战，都要用更大的代价去支付胜利，直到发现——自己本就是帝国延续的“备用帝座”。</strong></p>
</blockquote>
<hr>
<h2>0. 读感与叙事结构总纲</h2>
<p>全书用三层叙事质感叠加，保证“战锤味”与“剥洋葱式解谜”并存，同时把你的“外挂/叙事引擎”写成自然因果，而不是系统装置。</p>
<h3>0.1 三层叙事质感</h3>
<p>1）<strong>主线现场叙事（换壳推进）</strong>
每一卷的主线始终跟随“主意识”所在载体推进：战场奇观、政治残酷、宗教暴力、胜利代价。语言冷硬，但人物选择保留温度（不是全员疯批）。</p>
<p>2）<strong>多视角短章（结构显影）</strong>
在关键节点插入短章：底层矿奴、军团牧师、审判庭书记官、圣典注疏学者、舰队导航员、异端口供者……
这些短章有两种用途：</p>
<ul>
<li><strong>信息差钩子</strong>：读者看到“上层视角的隐语”，主线暂时不解释。</li>
<li><strong>回收证据链</strong>：后续主角获得相关载体/卷宗后，反证/印证短章。</li>
</ul>
<p>3）<strong>文书碎片（卷宗/战报/删改对照/口供）</strong>
每卷固定出现3—6段“文本碎片”，形式包括：</p>
<ul>
<li>《审判庭密卷摘录》</li>
<li>《远征军团战报》</li>
<li>《圣典注疏删改对照》</li>
<li>《异端口供》</li>
<li>《舰队航行黑匣记录》
<strong>关键：这些文本必须“推动剧情”，不是百科补丁。</strong>
它们要么给出行动指令、要么制造误导、要么留下删改痕迹逼主角推理，最后必须被剧情“换”出来，而不是一句神秘话糊弄。</li>
</ul>
<hr>
<h2>1. 世界观骨架：神权帝国与“虚潮”</h2>
<h3>1.1 宇宙的底色：虚潮（Warp感但不照搬）</h3>
<p>人类星海里存在一层与现实贴合的“<strong>虚潮</strong>”：</p>
<ul>
<li>它不是纯粹能量海，而是由<strong>情绪、愿望、执念、恐惧</strong>等“心象残响”构成的潮汐层。</li>
<li>航行必须穿越虚潮的“潮道”，否则星际距离无法在寿命内抵达。</li>
<li>虚潮里存在“<strong>潮灵</strong>”：它们不是固定形体的恶魔，而是从群体欲望里凝成的意志团块，会寄生信仰、吞吃愿望、诱发变异与疯狂。</li>
</ul>
<blockquote>
<p>这就把你的“愿望债/执念契约”天然钉在世界底层物理上：<strong>愿望不是隐喻，是一种能在虚潮留下“回声”的客观现象。</strong></p>
</blockquote>
<h3>1.2 帝国秩序：圣座帝国的三柱</h3>
<p>帝国统治不是靠单一军队，而是三柱互相牵制：</p>
<ul>
<li><strong>圣典院（教会）</strong>：解释真理、编纂圣典、赐予合法性。它能立圣人，也能宣人异端。</li>
<li><strong>审判庭（审判/异端系统）</strong>：对内肃清、对外监军，拥有“凌驾一切”的临时授权。它负责让恐惧成为秩序的一部分。</li>
<li><strong>远征军团（远征/殖民/清剿）</strong>：把帝国边界推向外海，把战火当作呼吸维持。</li>
</ul>
<p>此外还有一个第四势力常年游离但不可或缺：</p>
<ul>
<li><strong>机祷会（技术神权）</strong>：掌握舰船、圣械、灯塔阵列的维护权，把技术当作神迹，把维修当作祈祷。</li>
</ul>
<p>三柱彼此需要、彼此憎恶。<strong>战锤味的政治残酷就来自“体制内的结构性冲突”：</strong></p>
<ul>
<li>教会要“正确”，审判庭要“可控”，远征军团要“能打”。</li>
<li>三者同时成立时，普通人永远被碾。</li>
</ul>
<h3>1.3 帝国的“航标”：圣航灯与献祭税</h3>
<p>帝国能在虚潮里航行，靠的是遍布星海的“<strong>圣航灯阵列</strong>”：</p>
<ul>
<li>灯阵列以“祈祷回声”为燃料，维持潮道稳定。</li>
<li>每个行省都要缴纳“<strong>献祭税</strong>”：形式可能是灵能者、罪囚、征发的“祈祷团”、整城的“净化”。</li>
<li>民众被告知：这是“献身圣座”。</li>
<li>真相：这是维持现实不被虚潮吞没的“<strong>成本</strong>”。</li>
</ul>
<blockquote>
<p>这会成为贯穿全书的主题钉子：<strong>秩序靠罪维持，正义撕裂秩序。</strong></p>
</blockquote>
<hr>
<h2>2. 核心机制：愿债回响律（你的外挂/叙事引擎）</h2>
<p>这套机制必须“像世界法则一样自然”，并且严格遵守你定的硬限制。</p>
<h3>2.1 名称与表层理解（让世界里的人能讨论它）</h3>
<p>帝国官方禁词很多，所以各方会用不同的称呼：</p>
<ul>
<li>教会称：<strong>“圣愿应验”</strong>（神迹叙事）</li>
<li>审判庭内部密称：<strong>“回响附寄”</strong>（异常/武器叙事）</li>
<li>异端称：<strong>“愿潮寄生”</strong>（掠夺/献祭叙事）</li>
<li>机祷会称：<strong>“祈链跳转”</strong>（技术化叙事）</li>
</ul>
<p>主角最终会发现：这些说法都只说对了一部分。</p>
<h3>2.2 触发与宿主规则（硬限制固化）</h3>
<p><strong>规则A：不能自由选宿主</strong>
当主角死亡（或主意识脱壳）时，他会被虚潮的“回声牵引”拖走：</p>
<ul>
<li>牵引优先级取决于“<strong>附近濒死/已死者的执念强度</strong>”。</li>
<li>主角只能做极弱的“倾向引导”（例如把注意力集中在某个愿望主题上：复仇/救赎/生存），但<strong>不能精确点名某个人</strong>。</li>
<li>这保证潜入与破局不会变成随心所欲的作弊。</li>
</ul>
<p><strong>规则B：不强占活人</strong>
附寄只会发生在：
1）<strong>已死但尸体尚热</strong>（回声未散）
2）<strong>濒死者在最后意志里“自愿让位”</strong>（一种等价交换）
3）<strong>通过仪式明确同意</strong>（愿望契约）
活人不被强占，避免道德崩盘，也让“让位”本身成为戏剧冲突来源。</p>
<p><strong>规则C：载体上限与无敌感的来源</strong></p>
<ul>
<li>主角“本质”可能很硬（比如不易被虚潮吞、对某些圣械有亲和），但<strong>战斗表现严格由肉身决定</strong>。</li>
<li>无敌感来自两点：
1）他总能在关键节点“活下来并继续推进”（死不掉），
2）他在关键瞬间能“借用愿潮回声”做一次决定性压制（但代价巨大）。
他不是每具身体都能开满——很多时候他赢的是局面，不是战力碾压。</li>
</ul>
<p><strong>规则D：化身数量有限</strong></p>
<ul>
<li>初期：只能单载体。</li>
<li>中期：获得第二盏“命灯”（一个化身锚点），可同时在线2具载体，但主意识只能在一个载体里。</li>
<li>后期：最多3具（再多会失控），并且每多一具，身份磨损加速。</li>
</ul>
<p><strong>规则E：信息回流有延迟/成本</strong>
化身或旧载体经历的情报不会即时共享：</p>
<ul>
<li>必须在“<strong>同步节点</strong>”回收：例如审判庭的黑匣祷室、舰队的航行回声井、机祷会的铁祷共鸣台。</li>
<li>同步要付出代价：燃烧记忆、牺牲某段情绪、或让某个愿望“变硬成枷锁”。
这保证信息差持续存在，也为解谜提供“推理空间”。</li>
</ul>
<p><strong>规则F：愿望债与反噬（像任务但非系统）</strong>
每一次附寄/仪式都绑定一个“执念/愿望”：</p>
<ul>
<li>
<p>它不是弹窗，而是主角在新载体里会“被迫听见/感受”的<strong>愿望回声</strong>。</p>
</li>
<li>
<p>主角可以选择如何兑现（解释权在他），但不能完全无视。</p>
</li>
<li>
<p>若长期拖延或故意背弃，会触发<strong>反噬</strong>：</p>
<ul>
<li>身体机能异常、祈能失控、梦魇潮音、记忆断裂、附寄失败率上升。
这就是“长篇矛盾引擎”：每个宿主都带着债，债与主线冲突，逼主角做选择。</li>
</ul>
</li>
</ul>
<p><strong>规则G：身份磨损（终局倒计时）</strong>
兑现愿望越多，主角“自我边界”越薄：</p>
<ul>
<li>他会越来越像一个“由无数愿望拼成的器皿”。</li>
<li>读者能看到：他越来越冷、越来越像体制的工具，但偶尔会被某个微小愿望刺痛。</li>
<li>最终他会面临终局选择：成为“新的圣航灯核”/“新的帝座回声”或彻底断链，让人类自己面对黑暗。</li>
</ul>
<hr>
<h2>3. 主角与核心人物组（结构性矛盾的载体）</h2>
<h3>3.1 主角：主意识“烬”（内名）与第一躯壳“少女”</h3>
<ul>
<li>
<p>主角本体设定：<strong>普通穿越者</strong>，醒来时记忆残缺，只记得自己来自“无神、无审判庭”的世界。</p>
</li>
<li>
<p>他的“内名”可自取为“烬”：既是余火，也是燃料。这个名字只在内心用，外界永远以载体名字称呼他，强化“换壳叙事”的陌生感。</p>
</li>
<li>
<p>第一躯壳：<strong>一个少女</strong>（底层、弱小、却有最锋利的生存欲）。</p>
<ul>
<li>她的愿望很简单：“想活下去，想看见真正的天空。”</li>
<li>这会成为贯穿全书的“最小愿望钉子”：后面所有宏大叙事都要回到这个小愿望上，形成残酷反差。</li>
</ul>
</li>
</ul>
<h3>3.2 三条长期对线人物（不靠系统靠人）</h3>
<p>为保证300万字不散，设置三条贯穿全书的“长线对手/盟友”，他们每个人都代表体制的一种逻辑：</p>
<p>1）<strong>审判庭审判官·索勒姆（代号：黑铅）</strong></p>
<ul>
<li>理念：宁可错杀一亿，也要帝国活下去。</li>
<li>他会把主角当作“可控的异常武器”。</li>
<li>他不是纯反派：他能救星球，也能屠城。</li>
<li>长线冲突点：他要“控制主角的愿望解释权”，把主角变成审判庭工具。</li>
</ul>
<p>2）<strong>圣典院注疏总监·艾萨（代号：白页）</strong></p>
<ul>
<li>理念：真理必须统一，否则信仰崩塌，灯塔熄灭。</li>
<li>她擅长删改、重写、封存，把历史变成“可用的神话”。</li>
<li>她会试图“封圣”主角，把主角纳入圣典叙事。</li>
<li>长线冲突点：她要“定义主角是谁”，让他失去自我。</li>
</ul>
<p>3）<strong>远征军团军团长·赫塔尔（代号：铁誓）</strong></p>
<ul>
<li>理念：战场上只有胜利与代价，别拿圣典骗兵。</li>
<li>他敬畏圣座但更敬畏炮火。</li>
<li>他会在某些时刻成为主角最可靠的盟友，也可能在后期成为最危险的阻碍（当主角的选择威胁军团存续）。</li>
<li>长线冲突点：他要“可执行的答案”，而主角给不出无代价的答案。</li>
</ul>
<blockquote>
<p>这三人分别代表：<strong>审判（控制）/圣典（叙事）/远征（现实）</strong>。主角夹在中间，换壳只是表，真正推动长篇的是结构性冲突。</p>
</blockquote>
<hr>
<h2>4. 十卷（约300万字）叙事大纲：每卷都能重解释旧剧情</h2>
<p>下面按“卷”为单位规划（每卷约25—35万字），每卷都包含：</p>
<ul>
<li><strong>关键载体链</strong>（哪些壳、带来什么权限/信息）</li>
<li><strong>核心矛盾</strong>（外战+内斗）</li>
<li><strong>信息差钩子</strong>（丢术语、丢删改、丢矛盾证据）</li>
<li><strong>回收与重解释</strong>（让前文翻面）</li>
<li><strong>胜利代价</strong>（战锤味的核心）</li>
<li><strong>文书碎片建议</strong>（用卷宗推进剧情）</li>
</ul>
<hr>
<h1>卷一：灰城少女（“活下去”作为第一根钉子）</h1>
<h3>开场意象</h3>
<p>灰色蜂巢城的下层，空气像被烧焦的布。圣歌从扩音塔里一遍遍滚落：“愿望皆罪，唯信得赦。”
少女在垃圾与尸体之间醒来，手腕上有一圈被祈祷绳勒出的淤青。她不记得自己是谁，只记得“不能死”。</p>
<h3>关键载体链（无敌感的第一种形态：死而不止）</h3>
<p>1）<strong>少女·伊芙</strong>（第一载体）</p>
<ul>
<li>能力：几乎没有。只有敏捷、底层生存经验碎片、对恐惧的耐受。</li>
<li>信息：蜂巢城底层结构、净化队的巡逻、异端传闻。</li>
<li>愿望债：看见天空。</li>
<li>结局：她在一次净化行动中被误杀/献祭，主角第一次死亡。</li>
</ul>
<p>2）<strong>城防军新兵·马林</strong>（濒死让位）</p>
<ul>
<li>触发：伊芙死在战线边缘，附近一名新兵被潮疫撕裂，临死前想“至少别白死”。</li>
<li>能力：枪械、队列、最底层的军纪。</li>
<li>信息：城防军与审判庭的矛盾；净化行动的真实目的并非“治疫”，而是“封口”。</li>
</ul>
<p>3）<strong>审判庭随行书记官·拉姆</strong>（尸体尚热）</p>
<ul>
<li>触发：一场街区封锁后，书记官被流弹击中，抱着密卷死在祷室门口。</li>
<li>能力：识字、抄录、接触卷宗的权限。</li>
<li>信息差钩子源：主角第一次看到“删改痕迹”和“黑条遮蔽”。</li>
</ul>
<h3>核心矛盾</h3>
<ul>
<li>外部：蜂巢城爆发“潮疫”（虚潮渗入导致的群体变异）。</li>
<li>内部：审判庭要求“净化整区”，城防军想保住人口与产能，教会要求“举行大祈祷”。
三方都在说“为了活下去”，但活下去的定义不同。</li>
</ul>
<h3>信息差钩子（只给线索，不给解释）</h3>
<ul>
<li>“灰火”一词反复出现：净化火焰里偶尔会冒出灰白色火舌，烧人不烧物。</li>
<li>密卷编号：<strong>X-17《灰火异兆初录》</strong>（大量删改）。</li>
<li>祷告词被悄悄改过某几个字，改动后更像“献祭指令”而非“安抚祈祷”。</li>
</ul>
<h3>回收与重解释（卷尾翻面）</h3>
<p>卷末，主角在书记官载体里拼出第一块真相：</p>
<ul>
<li>“潮疫”并非自然灾害，至少这一次不是。</li>
<li>审判庭净化的重点不是治疫，而是<strong>掩盖某个更大的“渗漏源”</strong>。
而伊芙的死亡并非偶然——她被卷入了“渗漏源附近的献祭链”。</li>
</ul>
<h3>胜利代价</h3>
<p>主角用书记官权限偷出一份“封锁令”交给城防军，城防军因此提前撤离一批平民，保住一条撤离通道。
代价是：审判庭随后宣布该街区“全员疑似异端”，<strong>整区净化</strong>。
主角救下几千人，但更多人被烧成灰。
——无敌流的第一层：<strong>他能赢一场局部战，但救不了世界。</strong></p>
<h3>文书碎片建议（节选形式）</h3>
<ul>
<li>
<p>《审判庭密卷 X-17（节选）》：</p>
<ul>
<li>【——】处遮蔽“渗漏源”坐标</li>
<li>边注写着：“该词不可出现在公开祷告中。”</li>
</ul>
</li>
<li>
<p>《蜂巢城封锁令（删改对照）》：</p>
<ul>
<li>原文“隔离”被改成“净化”</li>
<li>原文“救治”被改成“献身”</li>
</ul>
</li>
</ul>
<p>卷尾钩子：主角在密卷夹层发现一个陌生印记——<strong>“第七码头·远征军团接应”</strong>，说明蜂巢城的事牵连到远征。</p>
<hr>
<h1>卷二：审判之印（从底层走进“审判机器”内部）</h1>
<h3>开场意象</h3>
<p>主角换到一具穿黑袍的躯壳里，第一次站在祷室的光里。
墙上挂着十二枚铁面具，代表审判庭十二座。面具下的人说：
“你见过灰火？那你就该学会闭嘴。”</p>
<h3>关键载体链（无敌感的第二种形态：权限压制）</h3>
<p>1）<strong>审判庭见习随员·莱因</strong>（濒死附寄）</p>
<ul>
<li>能力：基础审讯、枪械、少量“祈能”（用祷词稳定精神）。</li>
<li>权限：能跟随审判官进入封存区。</li>
<li>愿望债：证明自己不是“无用的人”，渴望被审判庭承认。</li>
</ul>
<p>2）<strong>异端窝点的“悔罪者”·乔瑟</strong>（自愿让位）</p>
<ul>
<li>触发：主角执行潜入任务时，目标悔罪者临死愿望是“让我把真话说出去”。</li>
<li>能力：熟悉异端网络、暗巷路线。</li>
<li>信息：异端并非铁板一块，他们也害怕“虚潮”，并且提到一个词：<strong>“灯塔在吃人”</strong>。</li>
</ul>
<p>3）<strong>教会祈火司祭·玛歌</strong>（尸体尚热）</p>
<ul>
<li>触发：一次公开净化仪式中，司祭被“灰火反噬”烧死。</li>
<li>能力：更完整的祈祷体系、对圣典的解释口径。</li>
<li>信息：教会内部也有“禁句”，有人在偷偷删词。</li>
</ul>
<h3>核心矛盾</h3>
<ul>
<li>
<p>外部：审判庭要拔除异端窝点，防止“渗漏源”扩大。</p>
</li>
<li>
<p>内部：审判庭分为两派：</p>
<ul>
<li><strong>净灭派</strong>：宁杀错不放过。</li>
<li><strong>归用派</strong>：异常也能当工具（把异端当诱饵，把潮疫当武器）。
主角被夹在两派斗争里：他作为“会换壳的异常”，既是证据也是诱饵。</li>
</ul>
</li>
</ul>
<h3>信息差钩子</h3>
<ul>
<li>“愿望债”第一次以世界内语言出现：审判官在审讯里说：
“人死时最硬的那句话，会在虚潮里留下回声。有人能听见，有人能被它拖走。”</li>
<li>审判庭的某个分支部门被提及：<strong>“归愿所”</strong>（但不解释是什么）。</li>
<li>主角发现自己每次换壳后，都会被一段“回声梦”追着走——这是愿望债的形式。</li>
</ul>
<h3>回收与重解释</h3>
<p>卷末主角拼出第二层真相：</p>
<ul>
<li>灰火不是单纯净化火，它像一种“回声燃烧”，能把愿望从人身上剥离。</li>
<li>审判庭内部有人在用灰火做实验——蜂巢城只是实验场之一。
这会重解释卷一的大净化：<strong>那不是失控，是试验后的“销毁现场”。</strong></li>
</ul>
<h3>胜利代价</h3>
<p>主角作为莱因潜入异端窝点，成功抓出“渗漏源”的搬运链，阻止一次更大规模潮疫扩散。
代价：为了堵住扩散，审判官下令把窝点所在街区连同数百无辜一起封进地下，点燃灰火。
主角亲手执行祷词，灰火从祈祷里冒出来，他第一次意识到：
<strong>自己也在成为机器的一部分。</strong></p>
<h3>文书碎片建议</h3>
<ul>
<li>《异端口供（节选）》：口供里反复出现“灯塔吃人”，但被审判官边注：“妄言，删。”</li>
<li>《归愿所内部备忘（遮蔽）》：提到“回响附寄者可作为……【——】”</li>
<li>《灰火使用规范（修订版）》：修订版增加“献词”，看似宗教，实则操作手册。</li>
</ul>
<p>卷尾钩子：主角被派往“第七码头”，登上远征舰队——理由是“把异常扔到战场上最安全”。</p>
<hr>
<h1>卷三：远征舰队（虚潮航行与战报叙事登场）</h1>
<h3>开场意象</h3>
<p>舰桥像一座漂浮的教堂，舷窗外是吞光的潮。
导航员戴着银盲眼，低声说：“灯亮了，我们就活；灯灭了，我们就变成潮里的祷词。”</p>
<h3>关键载体链（无敌感的第三种形态：战场奇观 + 关键压制）</h3>
<p>1）<strong>舰队警备军军士·布洛</strong></p>
<ul>
<li>能力：军纪、近战、舰内战斗。</li>
<li>信息：舰队纪律、远征军团的真实损耗。</li>
</ul>
<p>2）<strong>远征军团随军牧师·奥里安</strong>（自愿让位）</p>
<ul>
<li>触发：牧师在一次潮风暴中精神崩溃前，愿望是“让我的祷词继续被唱下去”。</li>
<li>能力：强祈能、能在战场上“稳定士气/压住潮音”。</li>
<li>权限：接触军团战报、与军团长近距离交流。</li>
<li>这是无敌流第一次“像样开挂”：他能用牧师载体在关键时刻压住潮音，救一支连队不崩。</li>
</ul>
<p>3）<strong>舰队导航副官·伊塔</strong>（尸体尚热）</p>
<ul>
<li>信息：虚潮航道的黑匣记录、灯塔信号的异常断续。</li>
</ul>
<h3>核心矛盾</h3>
<ul>
<li>外部：远征目标星系出现“潮裂”——现实被虚潮撕开，异形/异端混杂。</li>
<li>内部：舰队资源不足，军团长要求“速战速决”，审判庭要求“先封存证据”，教会要求“先立圣坛”。
谁先谁后，决定一整个星系的生死。</li>
</ul>
<h3>信息差钩子</h3>
<ul>
<li>战报里出现一个被涂黑的远征序列：<strong>“Ω序列”</strong>。</li>
<li>黑匣记录显示：某次航行中，灯塔信号并非来自帝国方向，而像是从“更深处”反射回来。</li>
<li>导航员提到禁忌词：“<strong>灯的背面有东西</strong>。”</li>
</ul>
<h3>回收与重解释</h3>
<p>卷末主角在导航副官载体里读到一段被封存的黑匣：</p>
<ul>
<li>灯塔并不只是“指路”，它在虚潮里像一个巨大的“<strong>吞口</strong>”。</li>
<li>吞口会吸走大量回声，才能维持航道。
这会重解释卷二异端的“灯塔吃人”——他们说的不是隐喻，而是事实的一角。</li>
</ul>
<h3>胜利代价</h3>
<p>主角用牧师载体在战场上压住潮裂，配合军团炮火打穿异端堡垒，赢下第一场真正意义上的“远征胜利”。
代价：为了稳定潮裂，舰队按照审判庭指令启动“祈祷阵列”，临时征调整舰平民舱的“祈祷团”——
<strong>一夜之间数千人失声、失忆、变成空壳。</strong>
军团活了，舰队秩序稳了，代价被写进战报一句话：
“祈祷团完成献身，士气稳定。”</p>
<h3>文书碎片建议</h3>
<ul>
<li>《远征军团战报·第七码头出航纪要》：用数字写残酷。</li>
<li>《舰队航行黑匣（节选）》：出现“反射信号”。</li>
<li>《牧师奥里安私人祷记》：写他如何被迫把祷词当武器。</li>
</ul>
<p>卷尾钩子：舰队接到命令，前往一颗被称为“<strong>献魂炉</strong>”的行省核心星——灯塔阵列的燃料中心之一。</p>
<hr>
<h1>卷四：圣典删改（真相开始“用文本换”）</h1>
<h3>开场意象</h3>
<p>一座行星级大教堂，穹顶是星图，地面是骨灰砖。
注疏学者们用羽笔写下神的名字，也用羽笔抹掉人的名字。</p>
<h3>关键载体链（无敌感的第四种形态：叙事权争夺）</h3>
<p>1）<strong>圣典院抄写员·费洛</strong></p>
<ul>
<li>能力：接触删改档案的最低门槛。</li>
<li>信息：删改对照、封存目录。</li>
</ul>
<p>2）<strong>注疏学者·德罗（“白页”派系成员）</strong></p>
<ul>
<li>权限：能进入“白页库”（帝国最敏感的原始史料库）。</li>
<li>愿望债：证明某段被删的历史“存在过”。</li>
</ul>
<p>3）<strong>审判庭驻教堂监察官·梅萨</strong></p>
<ul>
<li>信息：审判庭对圣典院的控制清单；谁被监视，谁将被“封口”。</li>
</ul>
<h3>核心矛盾</h3>
<ul>
<li>
<p>外部：献魂炉周边出现异端圣迹（“灰火自燃”），民众开始把某个死去的牧师奉为圣人。</p>
</li>
<li>
<p>内部：圣典院与审判庭争夺对“圣迹”的解释权：</p>
<ul>
<li>圣典院想封圣以稳定民心；</li>
<li>审判庭想封存以避免更多人追问“灰火是什么”。
主角夹在两者之间：他要活下去，也要弄清真相，但真相本身会引爆战争。</li>
</ul>
</li>
</ul>
<h3>信息差钩子（重点：删改对照）</h3>
<ul>
<li>主角在白页库看到“圣典原句”：
原句里，圣座并非“神”，而是“<strong>坐在灯下的王</strong>”。
现行版改成：“坐在光中的神”。</li>
<li>删改痕迹显示：改动发生在某次“Ω序列远征”失败之后。</li>
<li>白页库里还有一份“被撕掉的远征名单”，上面有一个奇怪的空位：
<strong>“第零随行者”</strong>。</li>
</ul>
<h3>回收与重解释</h3>
<p>卷末主角推理出第四层真相：</p>
<ul>
<li>帝国的“神权叙事”并非一开始就成立，它是某次巨大灾难后被强行统一的。</li>
<li>“第零随行者”这个空位，像是被从历史里抹去的人——但抹得并不干净。
这会让读者开始怀疑：主角的能力是否与“第零随行者”有关？（只埋不解）</li>
</ul>
<h3>胜利代价</h3>
<p>主角协助圣典院封圣，借封圣仪式制造“合法的混乱”，从而把白页库的一部分原始史料偷运出城。
代价：封圣引发狂热，审判庭借机清洗“疑似异端学者”，
<strong>整座注疏学院被净化，白页库被火封。</strong>
主角拿到真相碎片，但付出的是知识阶层的灭绝。</p>
<h3>文书碎片建议</h3>
<ul>
<li>《圣典注疏删改对照（节选）》：用两列对比制造冲击。</li>
<li>《封圣仪式流程（附审判庭边注）》：边注写着“净化时机”。</li>
<li>《白页库封存令》：密封条上印着审判庭与圣典院双印，暗示两者合谋。</li>
</ul>
<p>卷尾钩子：主角得知献魂炉的“燃料征发”即将升级——不再是祈祷团，而是整颗行省的“献界”。</p>
<hr>
<h1>卷五：献魂炉（胜利代价的核心卷：灯塔吃人被证实）</h1>
<h3>开场意象</h3>
<p>行省首都的地下，有一座巨大的炉。炉里没有火，只有祷词。
人被押进去，出来的是光。</p>
<h3>关键载体链（无敌感的第五种形态：知道真相仍能赢，但赢得更痛）</h3>
<p>1）<strong>献界罪囚·卡洛</strong></p>
<ul>
<li>能力：几乎无；但他能进入最底层的献界通道。</li>
<li>愿望债：活着见到女儿（典型小愿望钉子，刺穿宏大叙事）。</li>
</ul>
<p>2）<strong>机祷会铁祷师·诺因</strong>（濒死附寄）</p>
<ul>
<li>权限：接触炉心结构、灯塔阵列维护记录。</li>
<li>信息：灯塔确实以“回声”为燃料；献界不是宗教象征，是技术方案。</li>
<li>这是关键：技术文本把“神迹”剥皮，形成强烈反差。</li>
</ul>
<p>3）<strong>审判庭归愿所执行官·“剪枝者”</strong>（自愿让位或濒死）</p>
<ul>
<li>信息：归愿所的真正任务——把“异常回声”剪枝归用，防止虚潮反噬帝国。</li>
</ul>
<h3>核心矛盾</h3>
<ul>
<li>
<p>外部：潮裂扩大，献魂炉附近出现“潮灵实体化”。</p>
</li>
<li>
<p>内部：三方互咬：</p>
<ul>
<li>远征军团要炸炉封裂；</li>
<li>圣典院要继续献界维灯；</li>
<li>审判庭要封存真相、同时把主角这个异常抓进归愿所。</li>
</ul>
</li>
</ul>
<h3>信息差钩子（用亲历换真相）</h3>
<ul>
<li>主角亲眼看见献界流程：人被祷词“抽空”，留下躯壳。</li>
<li>灯塔阵列维护记录里出现“<strong>灯核衰减曲线</strong>”：说明灯核在“老化”。</li>
<li>归愿所提到：灯核老化时需要“更高质量的愿望”——即更强的灵能者、更大量的群体献界。</li>
</ul>
<h3>回收与重解释（全书第一次大翻面）</h3>
<p>卷末主角拼出第五层真相：</p>
<ul>
<li>帝国维持星海秩序的成本不是“偶尔牺牲”，而是一个不断加码的献祭体系。</li>
<li>灯核在衰减，意味着帝国走向“越献越不够”的死循环。
这会重解释前四卷的一切净化与删改：
<strong>不是为了纯粹的权力，而是为了隐藏“灯核衰减”的恐慌，防止帝国信仰崩塌。</strong></li>
</ul>
<h3>胜利代价（本卷标志性战锤段落）</h3>
<p>主角用铁祷师载体协助军团长在炉心外侧构建“临时稳态”，让军团能撤离献界通道的部分平民，避免献界规模扩大到整行省。
代价：为了维持稳态，他不得不启动一条“备用祈链”——
那条祈链连接的是一座孤儿院。
孤儿院被写进日志：
“备用祈链接入，灯光稳定，潮裂延缓。”</p>
<p>主角救了行省，代价是把“最像伊芙的那群孩子”送进炉里。
<strong>无敌感</strong>来自他赢下“行省不灭”的局面；
<strong>残酷感</strong>来自他必须用最小愿望去支付最大的秩序。</p>
<h3>机制升级：第二盏命灯的获得（但不是爽点，是枷锁）</h3>
<p>在炉心崩塌的混乱中，一位被封存多年的“未封圣者”（旧时代的牧师/战地圣人）濒死，愿望是：
“让我的名字别再被删掉。”
主角应愿，与其立下“命灯契”：</p>
<ul>
<li>主角获得第二个化身锚点（可同时在线2具载体）。</li>
<li>代价：那位未封圣者的“名”会一直在主角意识里燃烧，成为身份磨损的加速器。</li>
</ul>
<h3>文书碎片建议</h3>
<ul>
<li>《灯核衰减曲线（机祷会内部）》：冷冰冰的图表式描述（可用文字模拟）。</li>
<li>《献界名单（部分）》：名字被涂黑，只剩年龄与职业，更残酷。</li>
<li>《归愿所剪枝记录》：把人当枝条。</li>
</ul>
<p>卷尾钩子：审判庭正式给主角定性——不是人，也不是圣人，而是“可用的异常”。抓捕令发出。</p>
<hr>
<h1>卷六：两盏命灯（多线推进与信息回流成本正式成为剧情矛盾）</h1>
<h3>开场意象</h3>
<p>主角第一次同时“听见两具身体的呼吸”。
一具在审判庭的祷室里低声背诵禁句；
一具在军团的战壕里嚼着泥土。
他意识到：自己快分裂了。</p>
<h3>关键载体结构（两线但不失控）</h3>
<ul>
<li>载体A：<strong>审判庭线</strong>（用于卷宗、权力、封存区）</li>
<li>载体B：<strong>远征军团线</strong>（用于战场、奇观、兵权）
主意识只能在其中一个，但另一具可以作为“弱化化身”执行简单行动（且信息不同步）。</li>
</ul>
<h3>核心矛盾</h3>
<ul>
<li>外部：潮裂蔓延到邻近星系，出现“潮灾迁徙”。</li>
<li>内部：审判庭开始内斗：净灭派要杀主角，归用派要控制主角。</li>
<li>同时，军团长发现灯核衰减后，开始质疑圣典叙事，与教会冲突升级。</li>
</ul>
<h3>信息差钩子（用“不同步”制造悬念）</h3>
<ul>
<li>审判庭线看到的卷宗，军团线不知道；</li>
<li>军团线亲历的潮灾，审判庭线只能从战报看；</li>
<li>主角必须选择在哪个节点同步——而同步会燃烧某段记忆。
这会强迫主角做“叙事上的取舍”，也让读者体验信息差。</li>
</ul>
<h3>回收与重解释</h3>
<p>卷末主角发现：归愿所之所以盯上他，不只是因为他会换壳，而是因为——
<strong>他能“替愿”</strong>。
替愿意味着：他可以把某些即将变成潮灾的执念“引导成可控的回声”，相当于稳定虚潮的一种手段。
这会重解释：为何他总被牵引到“关键节点的濒死者”身上——不是巧合，而是虚潮把他当作“补丁”。</p>
<h3>胜利代价</h3>
<p>主角用两线配合，打赢一场审判庭追捕战：</p>
<ul>
<li>军团线掩护撤离，</li>
<li>审判庭线伪造卷宗，把追捕责任甩给异端。
代价：为了伪造成功，他必须“删掉”一段真实证据——
那段证据里，记录着某个被净化街区的无辜名单。
他亲手把他们从历史上抹去，换来自己继续推进主线。
<strong>这一步会让读者对主角产生复杂情绪：他不是纯白，他在学体制的手法。</strong></li>
</ul>
<h3>文书碎片建议</h3>
<ul>
<li>《双线战报对照》：同一事件两种叙事（军团战报 vs 审判庭纪要）。</li>
<li>《同步节点祷室记录》：同步要燃烧什么，用条目写清，形成硬约束感。</li>
</ul>
<p>卷尾钩子：主角在同步时听见第三个声音——来自虚潮深处，一个不属于任何宿主的“潮灵”。它在叫他“第零随行者”。</p>
<hr>
<h1>卷七：潮语者（主角被迫站到“混沌/虚潮”的视角）</h1>
<h3>开场意象</h3>
<p>他死在潮裂边缘，醒来时闻到甜腥的香。
这具身体在笑，笑得像哭。
有人在他耳边说：“欢迎回家，愿望的孩子。”</p>
<h3>关键载体链（高风险卷：落入敌方阵营但不等于立场转变）</h3>
<p>1）<strong>异端潮语者·维斯</strong>（濒死附寄，牵引失控）</p>
<ul>
<li>触发原因：潮裂附近，执念强度最高的往往是异端/疯子，主角无法选择。</li>
<li>能力：与潮灵沟通的“潮语”，可引发局部现实扭曲。</li>
<li>信息：异端如何理解灯塔、如何利用愿望。</li>
</ul>
<p>2）<strong>被献界者的空壳</strong>（短暂附寄）</p>
<ul>
<li>用来展示“没有愿望的人是什么样”：空、冷、像工具。</li>
<li>这会成为主角对自己终局的恐惧镜像。</li>
</ul>
<h3>核心矛盾</h3>
<ul>
<li>
<p>外部：潮灵准备借一次大规模“愿望洪峰”实体化，吞掉一座行省。</p>
</li>
<li>
<p>内部：异端阵营分裂：</p>
<ul>
<li>有人要彻底拥抱潮灵；</li>
<li>有人其实想摧毁灯塔（他们认为灯塔才是灾难源）。
主角必须在敌方视角里分辨“哪句疯话里藏着真话”。</li>
</ul>
</li>
</ul>
<h3>信息差钩子（本卷是“敌方证词”）</h3>
<ul>
<li>潮语者说：灯塔不是光，是**“钩子”**，把虚潮钩进现实。</li>
<li>潮灵说：人类的愿望本来会散去，但灯塔把它们集中起来，才滋养了潮灵。</li>
<li>异端提到一个古老名词：<strong>“帝座机”</strong>——不是神，是机器。</li>
</ul>
<h3>回收与重解释（第二次大翻面，重解释“灯塔吃人”）</h3>
<p>卷末主角拼出第六层真相：</p>
<ul>
<li>灯塔阵列确实在吃愿望，但它吃的不只是为了航行，它还在“<strong>压住某个更大的裂口</strong>”。</li>
<li>灯塔像一个塞子：拔掉它，现实会立即被虚潮淹没；不拔，它会越塞越紧，终有一天塞子碎裂，爆得更惨。
这让“推翻帝国/摧毁灯塔”不再是简单正义选项，复杂度暴涨，长篇升级空间打开。</li>
</ul>
<h3>胜利代价</h3>
<p>主角从异端阵营逃回，带回一份“帝座机”的线索。
代价：他逃回的方式是兑现潮语者的愿望——潮语者临死愿望是“让灯塔尝尝恐惧”。
主角把一段“恐惧回声”导入灯塔边缘节点，导致某个灯阵列短暂熄灭。
那一夜，三艘民船迷航，数万人失踪。
他带回关键真相，但也亲手制造了灾难。
<strong>无敌流的残酷升级：主角越接近真相，越像灾难本身。</strong></p>
<h3>文书碎片建议</h3>
<ul>
<li>《潮语者祭词（异端文本）》：看似胡言，藏关键隐喻。</li>
<li>《灯阵列短暂熄灭事故通告》：官方解释与真实原因矛盾。</li>
<li>《失踪民船名单》：把代价落地成“人”。</li>
</ul>
<p>卷尾钩子：审判庭内战公开化，净灭派宣布“归愿所为异端工具”，准备清洗；归用派宣布“主角为圣座赐下的回声武器”，准备强行收编。</p>
<hr>
<h1>卷八：审判庭内战（体制撕裂：胜利等于帝国变弱）</h1>
<h3>开场意象</h3>
<p>审判庭的十二面具碎了三面。
碎面具下的脸更像人，也更像鬼。
他们说：圣座需要一个新的“解释”，而解释需要血。</p>
<h3>关键载体链（权力层上升：从执行者变成决策者）</h3>
<p>1）<strong>审判庭高阶审判官·索勒姆（或其亲信）</strong></p>
<ul>
<li>主角可能短暂落位到其亲信/副手身上，获得接触内战核心的视角。</li>
<li>重点不在爽，而在“看见权力如何制造合理的罪”。</li>
</ul>
<p>2）<strong>圣典院白页派核心·艾萨（白页）</strong></p>
<ul>
<li>主角未必直接附寄她（活人不强占），但可能通过她身边濒死的抄写官获取“白页派”的核心策略：封圣主角，统一叙事。</li>
</ul>
<p>3）<strong>军团长赫塔尔（铁誓）阵营的参谋官</strong></p>
<ul>
<li>让主角从军方视角看“审判庭内战会害死多少前线士兵”。</li>
</ul>
<h3>核心矛盾</h3>
<ul>
<li>外部：潮灾前线需要统一指挥。</li>
<li>内部：审判庭内战导致指挥链断裂，远征军团开始“不听令”。
教会趁机扩张封圣运动，企图把主角纳入“圣愿使”体系。
主角被迫选择：</li>
<li>被审判庭控制成为武器；</li>
<li>被教会封圣成为旗帜；</li>
<li>或与军团结盟成为叛逆。</li>
</ul>
<h3>信息差钩子</h3>
<ul>
<li>
<p>“帝座机”线索被不同派系抢夺，各自解释不同：</p>
<ul>
<li>净灭派说那是异端谣言；</li>
<li>归用派说那是最高机密；</li>
<li>白页派说那是“不可言说的圣迹”。</li>
</ul>
</li>
<li>
<p>读者会在不同文本碎片里看到互相矛盾的“真相版本”，逼着自己推理。</p>
</li>
</ul>
<h3>回收与重解释</h3>
<p>卷末主角确认第七层真相：</p>
<ul>
<li>审判庭之所以能凌驾一切，是因为它掌握“灯塔稳定权”的部分钥匙。</li>
<li>内战的本质是：谁来掌握“献祭税”的调度权，谁就掌握帝国生死。
这重解释了早期审判庭对地方的残酷：那不是单纯暴政，而是“燃料调度”的暴力形式。</li>
</ul>
<h3>胜利代价</h3>
<p>主角通过政治操作让内战短期停火，形成“前线共同体”。
代价：他必须把一个派系推出来当替罪羊——归愿所被宣判为异端机构，核心成员公开处决。
其中有几人曾在卷二、卷五里给过主角关键帮助。
主角赢下停火，但切断了自己与“机制真相”的一部分活人证据链。
他离终局更近，也更孤立。</p>
<h3>文书碎片建议</h3>
<ul>
<li>《审判庭内战公告》：官方语言的冷酷。</li>
<li>《处决名单》：把代价具象化。</li>
<li>《白页派封圣草案》：用“温柔语言”包裹暴力。</li>
</ul>
<p>卷尾钩子：军团长带着主角的线索，决定进军帝国核心——去“圣座界”，亲眼确认“帝座机”是否存在。</p>
<hr>
<h1>卷九：帝座阴影（抵达核心：神话被剥到骨头）</h1>
<h3>开场意象</h3>
<p>圣座界不是一颗星球，是一座巨大的、覆盖行星的祭坛城市。
天空没有星，因为灯太亮。
亮得像在掩盖什么。</p>
<h3>关键载体链（接触终局真相：帝座机/备用灵核）</h3>
<p>1）<strong>机祷会核心维护师·“主钥”</strong>（濒死附寄）</p>
<ul>
<li>
<p>权限：进入帝座机外围维护廊道。</p>
</li>
<li>
<p>信息：帝座机的真实结构：</p>
<ul>
<li>灯核、祈链、献魂炉群、回声稳定器……</li>
<li>全部是工程学与神权语言的混合。</li>
</ul>
</li>
</ul>
<p>2）<strong>圣典院最高注疏者的随侍</strong></p>
<ul>
<li>信息：最高层如何用“叙事”压住恐慌。</li>
</ul>
<p>3）<strong>帝座禁卫（或其后勤军医）</strong></p>
<ul>
<li>信息：帝座界真实的“死亡规模”，以及禁卫对真相的沉默。</li>
</ul>
<h3>核心矛盾</h3>
<ul>
<li>
<p>外部：灯核衰减进入不可逆阶段，潮灾逼近核心。</p>
</li>
<li>
<p>内部：各派系聚集圣座界，准备终局方案：</p>
<ul>
<li>归用派想把主角接入帝座机，成为新的回声稳定器；</li>
<li>白页派想让主角“登圣”，以神话替换恐慌；</li>
<li>军团派想炸掉部分献魂炉，减少献祭规模，哪怕引发短期崩坏。
主角第一次被迫面对：<strong>自己是否愿意成为“新的灯核”</strong>。</li>
</ul>
</li>
</ul>
<h3>终局真相拼图（第八层）</h3>
<p>主角在帝座机维护记录里读到一条被封存的古老工程备注：</p>
<ul>
<li>帝座机最初设计就包含“备用灵核方案”，以防主灯核崩解。</li>
<li>备用灵核的核心要求不是技术能力，而是：
<strong>“能承载并解释大量愿望回声而不立刻崩溃的意识容器。”</strong>
这就是主角。
他不是偶然穿越，而是某次灯核异常时被“虚潮牵引”出来的补丁。
他的换壳能力，是系统的副作用，是帝座机对虚潮的“协议漏洞”。</li>
</ul>
<blockquote>
<p>这会重解释全书的牵引规律：为什么他总在关键节点落位、为什么愿望债像链条一样拖着他——因为帝国这台机器在把他一步步“校准”成备用零件。</p>
</blockquote>
<h3>胜利代价</h3>
<p>主角为了阻止各派系当场内战导致灯核崩溃，选择暂时接受“接入测试”，用自己的回声能力短暂稳定灯核。
代价：他在接入中失去一段“最像人”的记忆——可能是他穿越前的某个重要人/事，或卷一少女伊芙的那句“想看天空”被磨损成模糊。
读者将第一次真切感到：
<strong>终局不是死，而是“我是谁”的消失。</strong></p>
<h3>文书碎片建议</h3>
<ul>
<li>《帝座机维护日志（封存级）》：用工程语写神权恐怖。</li>
<li>《备用灵核方案（删改对照）》：删掉“备用”二字，改成“圣意”。</li>
<li>《禁卫沉默誓约》：解释为何最高层没人反抗。</li>
</ul>
<p>卷尾钩子：潮灾冲破外环，帝座界将进入“光尽之日”。主角获得第三盏命灯的可能性出现——但代价会是不可逆的非人化。</p>
<hr>
<h1>卷十：光尽之日（终局：赢不赢都要付出更大的代价）</h1>
<h3>开场意象</h3>
<p>灯开始闪烁。
每闪一次，就有一座城的人跪下祈祷；每祈祷一次，就有一批人被押进炉。
人类用恐惧点亮灯，用灯照亮恐惧。</p>
<h3>关键结构：三盏命灯的终局布阵（但绝不爽，只有艰难）</h3>
<ul>
<li>命灯A：帝座界内部（接入/稳定/政治）</li>
<li>命灯B：前线潮裂（战场/封裂/奇观）</li>
<li>命灯C：圣典叙事中枢（删改/封圣/舆论）
主角必须利用三线不同步制造“时间差”，让不同派系误以为自己赢了，从而争取封裂窗口。</li>
</ul>
<h3>终局矛盾：三种方案，没有一种干净</h3>
<p>1）<strong>成为新的灯核（最大稳定）</strong></p>
<ul>
<li>帝国延续，潮灾暂退。</li>
<li>代价：主角自我彻底被磨成“愿望容器”，成为新的帝座回声。</li>
<li>这是体制的胜利，也是人的死亡。</li>
</ul>
<p>2）<strong>炸掉献魂炉群（减少献祭）</strong></p>
<ul>
<li>短期内潮灾可能暴涨，帝国大量航道崩溃。</li>
<li>代价：无数行省会在黑暗中独自灭亡。</li>
<li>这是正义的尝试，但会撕裂秩序。</li>
</ul>
<p>3）<strong>断开灯塔协议（彻底断链）</strong></p>
<ul>
<li>人类失去虚潮航行能力，帝国瓦解为孤岛。</li>
<li>代价：文明退化，但虚潮也可能因失去集中“愿望燃料”而逐渐平静（不确定，必须用剧情推理与实验验证）。</li>
<li>这是革命，但代价是“你愿不愿意让未来的人不再活在灯下”。</li>
</ul>
<h3>终局揭秘（第九层，也是最后一层）</h3>
<p>主角在最后一次同步节点里听见“帝座”的原始回声——不是神谕，而是一句疲惫的人话：
“我点灯，是因为我不点灯，你们就活不到明天。”
这句话会把“帝国的罪”与“帝国的必要”压成一块石头：
<strong>不是谁坏，而是世界坏。</strong></p>
<h3>最终胜利代价（可选终局走向，但都符合战锤味）</h3>
<p>给你三种结局模板（都能让读者心口发紧），你后续可按偏好选一条：</p>
<p><strong>结局A：烬入灯（悲壮稳定）</strong>
主角选择成为灯核，帝国延续。
最后一幕：新灯亮起，天空终于“像白昼”，但主角再也想不起“天空为何值得看”。
封圣仪式上，圣典院写：
“圣愿使自愿登圣。”
审判庭写：
“异常已归用。”
远征军团写：
“战役胜利。”
只有读者知道：那是一个普通人被磨成零件。</p>
<p><strong>结局B：灯碎人散（革命撕裂）</strong>
主角炸掉献魂炉群，灯塔暗下去，帝国碎裂。
潮灾吞掉一部分行省，另一部分在黑暗中苟活。
最后一幕：卷一那句愿望回响回来——某个孩子在废墟里抬头说：
“我看见星了。”
代价巨大，但“天空”终于不是灯的颜色。</p>
<p><strong>结局C：协议重写（最难、最“玄鉴式”）</strong>
主角不选极端，而是在三线布阵中“重写灯塔协议”：</p>
<ul>
<li>让灯塔不再以“剥夺愿望”为燃料，而是以“自愿回声”与“可逆同步”为代价。</li>
<li>这需要牺牲主角自身的一部分作为协议担保（例如永远无法再回到完整自我，但保留一点“人”的核）。
结局不完美：帝国仍残酷，但献祭规模下降，真相开始流通。
最后一幕：白页库重新开门，第一行未删的原句被写回：
“坐在灯下的王。”
神话退一步，人性进一寸。</li>
</ul>

<h2>5. 全书“剥洋葱”谜团清单与回收节奏（保证每阶段重解释）</h2>
<p>为了确保“每一阶段旧剧情都能被重解释”，把谜团按层级设计，并规定回收点：</p>
<h3>第一层（卷1回收一部分）</h3>
<ul>
<li>灰火是什么？潮疫为何发生？
回收：灰火不是单纯净化，是回声燃烧。</li>
</ul>
<h3>第二层（卷2—3回收）</h3>
<ul>
<li>归愿所是什么？为什么盯主角？
回收：归愿所剪枝异常回声，把异常当工具。</li>
</ul>
<h3>第三层（卷3回收）</h3>
<ul>
<li>灯塔信号为何反射？
回收：灯塔不是单向“照”，它像吞口/钩子。</li>
</ul>
<h3>第四层（卷4回收）</h3>
<ul>
<li>圣典为何删改？“第零随行者”是谁？
回收：神权叙事是灾难后强行统一，历史被人为抹除。</li>
</ul>
<h3>第五层（卷5大翻面）</h3>
<ul>
<li>献祭税的真相？
回收：灯核衰减死循环，献界是工程方案。</li>
</ul>
<h3>第六层（卷7大翻面）</h3>
<ul>
<li>灯塔到底是救命还是灾难？
回收：灯塔像塞子，拔掉会立刻淹没，不拔会越塞越炸。</li>
</ul>
<h3>第七层（卷8回收）</h3>
<ul>
<li>审判庭内战的本质？
回收：争夺献祭调度权与灯塔稳定权。</li>
</ul>
<h3>第八层（卷9回收）</h3>
<ul>
<li>主角为何是主角？
回收：备用灵核方案，主角是协议漏洞的补丁。</li>
</ul>
<h3>第九层（卷10终局回收）</h3>
<ul>
<li>帝座是神还是人？
回收：帝座回声是人的疲惫选择，世界逼出罪。</li>
</ul>

<h2>6. 写作执行的“硬约束清单”（保证不崩、不断供）</h2>
<p>把你定的硬限制进一步“写作化”，让每卷都能自检：</p>
<p>1）<strong>每次换壳必须有“牵引理由”</strong>：附近濒死者执念最强是谁？主角为何会落到这具躯壳？
2）<strong>每次附寄都要带一个愿望债</strong>：愿望要具体、可执行、会与主线冲突。
3）<strong>每卷至少一次“同步节点”戏</strong>：用同步成本制造选择与悬念。
4）<strong>每卷至少一次“胜利代价”落地到人名</strong>：不是抽象死伤，是具体名单/职业/年龄。
5）<strong>每卷必须有一处“文本碎片推动剧情”</strong>：它要么改变行动、要么制造误导、要么提供证据链。
6）<strong>无敌感只在关键时刻释放</strong>：平时受载体限制吃瘪，关键时刻用权限/祈能/愿潮压制局面，但代价回收。
7）<strong>身份磨损可视化</strong>：每卷让主角失去/改变一小块“人性习惯”（比如对某种气味、某句口头禅、对天空的执着），到终局形成恐惧。</p>

<h2>7. 你可以直接开写的“卷一前五章章纲”（示范叙事落地，不用系统也能像任务）</h2>
<blockquote>
<p>这里给你一个可直接开篇的“信息差钩子+愿望债+审判味”组合模板。</p>
</blockquote>
<h3>第1章：灰层醒来</h3>
<ul>
<li>少女伊芙在垃圾层醒来，听见远处圣歌。</li>
<li>读者不知道她是谁，只看到她被祈祷绳勒过的手腕。</li>
<li>她的脑中闪过一句不属于她的中文（穿越钩子），但立刻被圣歌压下去（世界压住个人）。</li>
</ul>
<h3>第2章：净化队</h3>
<ul>
<li>净化队来“清理潮疫”，实际上在找“渗漏源附近的活口”。</li>
<li>伊芙逃跑，撞见一个濒死的矿工，他嘴里重复：“别让他们拿走我最后一句话。”</li>
<li>主角第一次听见“愿望回声”的质感，但不解释。</li>
</ul>
<h3>第3章：天空在上面</h3>
<ul>
<li>伊芙躲进废弃升降井，抬头看到一条极细的光缝（天空钩子）。</li>
<li>她的愿望成形：<strong>想看见真正的天空</strong>。</li>
<li>同时，读者看到一段插入文本：《蜂巢城封锁令（节选）》——“隔离/净化”字样删改对照。</li>
</ul>
<h3>第4章：灰火</h3>
<ul>
<li>净化队点火，火里冒出灰白色火舌，烧人不烧物。</li>
<li>伊芙被误当异端，灰火舔上来。</li>
<li>她临死那一瞬，愿望不是宏大救赎，只是：
“让我再看一眼上面。”</li>
<li>主角第一次死亡，触发附寄。</li>
</ul>
<h3>第5章：新兵马林</h3>
<ul>
<li>主角醒在新兵马林体内，腹部开裂，血流不停。</li>
<li>马林的愿望是“别白死”，他把枪塞给主角，等于自愿让位。</li>
<li>主角第一次意识到：这不是复活，这是<strong>被拖着替别人把话说完</strong>。</li>
</ul>
