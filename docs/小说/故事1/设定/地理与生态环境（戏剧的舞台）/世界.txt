<h2>0.1 核心光源（人类认知中的"恒星"）</h2>
<p>人类观测到的核心光源<strong>外观上呈现为一颗老年、相对安静的 M 型红矮星</strong>——光谱偏红、输出稳定、耀斑活动极少。但人类并不知道真相：这颗"恒星"实际上是一枚被封装的<strong>恒星胚胎</strong>，外包裹着看护者建造的<strong>辐射幕</strong>。辐射幕让胚胎的能量输出呈现为类恒星光球，但其光谱比自然恒星更窄、漂移更小、活动更受抑制。人类只把它当作一颗异常稳定的红矮星，从未意识到它是一个被刻意维持在"既不孵化也不衰亡"状态的胚胎。</p>
<ul>
<li>质量：(M_* = 0.25,M_\odot)</li>
<li>光度：(L_* = 0.008,L_\odot)</li>
<li>适居带标称半径（接近地球辐照）：
[
a_\mathrm{hab} \approx \sqrt{\frac{L_*}{L_\odot}},\mathrm{AU}
=\sqrt{0.008},\mathrm{AU}\approx 0.089,\mathrm{AU}
]
数值：(a_\mathrm{hab}\approx 1.33\times 10^{10},\mathrm{m}\approx 1330) 万 km</li>
</ul>

<p>在 (a=0.089,\mathrm{AU}) 的圆轨道上（这是 Layer C 主栖居带所在）：</p>
<ul>
<li>轨道圆周：
[
C = 2\pi a \approx 8.37\times 10^{10},\mathrm{m}

<li>轨道速度：

v=\sqrt{\frac{GM_*}{a}}\approx 49.9\ \mathrm{km/s}

<li>轨道周期（“自然年”）：

T= \frac{2\pi a}{v}\approx 19.4\ \text{天}

<h2>Layer A：内侧推进与控光帽（Engine / Cap Zone）</h2>
<h3>A1. 结构与位置（“帽子”不是一块，是一片“薄膜群岛海”）</h3>
<p><strong>空间位置建议：</strong>(r=0.015)–0.03 AU（比主栖居带更靠近恒星，能量密度高，利于做遮蔽与光路整流）</p>
<ul>
<li>在 0.02 AU 处，辐照度约 27.2 kW/m²，平衡温度（反照率 0.3）约 538 K。
这意味着“帽子”材料必须是高反射、耐热、可自修复的薄膜/晶格结构。</li>
</ul>
<p>**结构单元：**不是“刚体镜”，而是三类单元混编：</p>
<ol>
<li>
<p><strong>反射帆单元（Mirror-sail tiles）</strong></p>
<ul>
<li>主要作用：把恒星辐射在宏观上“偏向”某些方向（形成辐射各向异性）</li>
<li>兼任：把可用光导入 Layer B 的配光/输能路由</li>
</ul>
</li>
<li>
<p><strong>遮光/屏蔽单元（Shade tiles）</strong></p>
<ul>
<li>主要作用：耀斑/高能粒子事件时，为外层提供短时强遮蔽</li>
<li>兼任：让外层的“昼夜/季节”具备硬开关能力</li>
</ul>
</li>
<li>
<p><strong>控制与通信骨架（Spine relays）</strong></p>
<ul>
<li>稀疏分布的“硬节点”，提供高可靠姿态参考、时钟、束路指令播发</li>
<li>它们是 Layer A 在废弃后最可能仍“以残响形式运转”的部分</li>
</ul>
</li>
</ol>
<h3>A2. 推进的物理上限（光子动量）与“为什么会很慢”</h3>
<p>如果你只靠“让恒星辐射不对称”来推进，推力上限量级是：
[
F \sim \eta\frac{L_*}{c}
]
其中 (\eta) 是“净动量整流效率”（0 到 1 之间；越接近 1，越像把光当火箭喷流）。</p>
<p>对本恒星：</p>
<ul>
<li>(\frac{L_*}{c}\approx 1.02\times 10^{16}\ \mathrm{N})</li>
<li>恒星质量 (M_*\approx 0.25M_\odot\approx 4.97\times 10^{29}\ \mathrm{kg})</li>
<li>加速度上限（(\eta=1)）：
[
a_\mathrm{photon}\approx \frac{L_<em>}{M_</em>c}\approx 2.05\times 10^{-14}\ \mathrm{m/s^2}
]
这非常小，但它的叙事价值在于：<strong>它永远推、推到文明忘了它在推</strong>。</li>
</ul>
<blockquote>
<p>你可以把“完好期的恒星方舟”写成：</p>
<ul>
<li>平时是“巡航态”（(10^{-14}) 量级，几乎不可感知）</li>
<li>关键时期才启用“主动恒星发动机”（下面会给一个合理的工程化表达），把加速度抬到 (10^{-10}\sim 10^{-9}\ \mathrm{m/s^2}) 量级，用于改航向、避让、迁徙窗口。</li>
</ul>
</blockquote>

<p>你前面强调“不要绳索”，那 Layer A 的镜阵要么是：</p>
<ul>
<li><strong>statite（光压平衡引力的准静悬单元）</strong></li>
<li>要么是<strong>轨道单元 + 主动维持（微推力/电推进）</strong></li>
</ul>
<p>对“理想反射 statite”单元，平衡条件推得一个非常重要的结论：
它能否悬停<strong>只取决于单位面积质量（面密度）</strong>，与距离无关。</p>
<p>[
\sigma_\mathrm{statite}=\frac{L_<em>}{2\pi GM_</em>c}
]</p>
<p>代入本恒星，得到：</p>
<ul>
<li>(\sigma_\mathrm{statite}\approx 4.9\times 10^{-5}\ \mathrm{kg/m^2}\approx 0.049\ \mathrm{g/m^2})</li>
</ul>
<p>这意味着什么？</p>
<ul>
<li>若你希望单元“纯靠光压悬停”，每平方米只能有 <strong>50 毫克级</strong>的质量。</li>
<li>这对应“几十纳米量级”的超薄反射膜（还要能抗撕裂、抗微陨石、能自愈）。</li>
<li><strong>所以工程上更合理的写法是：</strong>
Layer A 是“以光压为主的减载 + 以微推力为补偿的准静悬阵列”。
完好期靠控制系统把它压成一个“帽”；废弃后控制断裂，它就会变成“漂移的薄膜群岛”，遮蔽与反射图案长期偏置。</li>
</ul>
<blockquote>
<p>这条物理约束在叙事上非常好用：
<strong>你不需要写“帽子突然消失”，你只要写它“越来越散、越来越偏、越来越不准”。</strong>
居民看到的就是气候带固化、日照走样、神话化的天命分区。</p>
</blockquote>

<h2>Layer B：戴森云采能与配光网（Power / Light Swarm）</h2>

<h3>B2. 能量账：你要写“世界”，就必须知道它靠什么供能</h3>
<p>在适居带，天然辐照约 1375 W/m²。
如果你只靠“自然入射”也能活，但你要的是：</p>
<ul>
<li>人工昼夜（准时开关）</li>
<li>人工季节（调强弱、调光谱）</li>
<li>工业与推进（高功率）</li>
<li>长期稳定（灾害时切换）</li>
</ul>
<p>所以 Layer B 必须有“远超生态基本需求”的供能冗余。</p>
<p>一个叙事上好用、计算又不夸张的设定是：</p>
<ul>
<li>
<p>每个“主弧段”（1°）在完好期拥有可调的输入功率：(10^{14}\sim 10^{15}\ \mathrm{W}) 级</p>
<ul>
<li>(10^{14}\ \mathrm{W}) 足够支撑一个超高工业化的大区（对比：人类当代文明总功率 ~(10^{13}\ \mathrm{W})）</li>
<li>(10^{15}\ \mathrm{W}) 则是“重工业+船坞+束路推进”的级别</li>
</ul>
</li>
</ul>
<p>对应需要多大的采能面积？
在 0.05 AU 处辐照约 4356 W/m²。假设采能与输能总效率 30%（包括转换、传输、接收）：</p>
<ul>
<li>每获取 (10^{14}\ \mathrm{W}) 的可用功率，需要收集：
[
A \approx \frac{10^{14}}{0.3\times 4356}\approx 7.65\times 10^{10}\ \mathrm{m^2}
]
即约 7.65 万 km²（相当于一个大国的面积量级，但在太空薄膜里并不离谱）。</li>
<li>若是 (10^{15}\ \mathrm{W})，面积乘 10。</li>
</ul>
<blockquote>
<p>这给你一个非常清晰的“世界观体量”：
<strong>每个弧段背后都有一个“看不见的采能领地”。</strong>
完好期它是电网；废弃期它是“天上不可触碰的银色神田”。</p>
</blockquote>

<h3>C1. 1° 弧段的"主世界"形态：单一弧段栖居体</h3>
<p>为了实现"居民真的把它当世界"，主居住体最好满足：</p>
<ul>
<li>面积足够大，能承载完整生态与国家级文明</li>
<li>天空观感可控（能隐藏太空特征）</li>
<li>世界边界可神话化（边墙/禁山）</li>
<li>废弃后仍可在局部自洽生存（自治、分区、残存系统）</li>
</ul>
<p>每个 1° 弧段是一个<strong>单一的连续栖居体</strong>，像一片漂浮的大陆，弧长约 23 万公里，建造在锁定环表面之上。弧段两端以端盖框架封闭，弧段内部是连续的大气层、海洋与陆地。</p>
<p><strong>天壁（边墙）高度：80–150 km</strong></p>
<p>天壁是弧段边缘的高墙结构，承担三项功能：</p>
<ol>
<li><strong>大气边界</strong>：阻止气体向外泄漏。在 0.8g、常温（~288K）、以 N₂ 为主的大气中，尺度高度约 10 km。墙顶气压降到 1 mbar 需约 70 km，降到 1 µbar 需约 140 km。80–150 km 的墙高足以让墙顶接近真空。</li>
<li><strong>视野遮蔽</strong>：不让居民轻易看到上方的真空与结构，维持"世界感"。</li>
<li><strong>禁区神话化</strong>：墙顶成为神话与禁忌的来源。</li>
</ol>
<p><strong>墙顶重力</strong>：在旋转重力下，重力随高度近似线性下降：g(h) ≈ g₀(1 − h/R)。若 R = 1330 万 km、h = 100 km，则 g_top ≈ 0.99993 g₀，几乎无差异。这意味着"登壁仪式/禁山试炼"的体感差异主要来自气压下降与温度骤降，而非重力变化。</p>
<p>弧段上方维护一套"<strong>天空散射层（Sky-veil）</strong>"：高空气溶胶与漂浮薄膜的混合层</p>
<ul>
<li>它让天空呈现稳定蓝色</li>
<li>抑制真实星空可见度（隐藏自转导致的天幕旋转）</li>
<li>也是微陨石防护与辐射屏蔽的一部分</li>
<li>废弃后逐渐破损 → <strong>星星开始"回归"</strong>（超级强的伏笔/揭示机制）</li>
</ul>
<blockquote>
<p>这一步很关键：
如果你完全开放天空，旋转栖居体会让星空在一天内转一圈，太容易"露馅"。
你要"长期误认世界"，就需要一个能把天空"自然化"的散射层/天幕系统。</p>
</blockquote>
<h3>C2. 弧段栖居体的核心物理量级</h3>
<p>环带主轨道半径 R ≈ 1330 万公里，环带以自转提供类地重力，自转周期约 2–3 天。</p>
<ul>
<li>角速度（以 2.7 天周期为例）：
ω ≈ 2.7×10⁻⁵ rad/s</li>
<li>边缘线速度：
v = ωR ≈ 361 km/s</li>
<li>离心加速度（产生重力）：
g = ω²R ≈ 9.7 m/s²（接近 1g）</li>
</ul>
<p><strong>可写成世界观细节：</strong></p>
<ul>
<li>这世界的"自然天象"若不被天幕遮蔽，会出现"天穹缓慢旋转"（2–3 天一圈）；</li>
<li>因为有天幕，居民只看到"太阳按 24 小时制走"，并把偶尔出现的真实星点当神迹。</li>
</ul>
<p><strong>大气质量量级（写资源与灾害）：</strong>
单个弧段面积：弧长约 23 万公里，取平均宽度 1 万公里：
A ≈ 2.3×10⁵ × 10⁴ km² ≈ 2.3×10⁹ km² = 2.3×10¹⁵ m²</p>
<p>1 atm 的大气柱质量约 P/g ≈ 1.03×10⁴ kg/m²，则：
M_atm ≈ 2.4×10¹⁹ kg（单个弧段）</p>
<p>这是可由彗星/冰小行星供给、也可被"世界管理层"长期维护的量级。
废弃后：漏气、补气失败、分区封闭会成为"末世工程"。</p>

<h3>C4. 场景切片（完好期 vs 废弃期）</h3>
<p><strong>完好期：从枢纽脊港落地（重力制度的切换）</strong></p>
<blockquote>
<p>飞船停在非旋转的枢纽脊港，脚下是零重力的金属走廊。你沿着"落井"进入转运电梯，警示灯提示：即将进入 1g 区域。门开的一瞬间，世界有了方向——风有了重量，人的步伐重新被地面接住。你抬头看到蓝天，太阳盘像被精细打磨过的白金圆，远处天壁如同永恒山脉，高度超过 80 公里，顶端消失在云层之上。你会忘记自己刚才还漂浮。</p>
</blockquote>
<p><strong>废弃期：农夫的日落（太阳不是落下去，是暗下去）</strong></p>
<blockquote>
<p>太阳在黄昏并不滑向地平线，它只是逐渐熄灭，像灯芯收火。云影的移动也像被谁在幕后推动：每一天都几乎相同。孩子问"太阳住在哪里"，大人说住在天壁之后。没人知道太阳其实住在日照桅杆的扩散镜上；而天壁之后，确实有通道——只是那通道通向零重力的骨架与真空。</p>
</blockquote>
<hr>
<h2>Layer D：外层散热与船坞带（Radiator / Yard Belt）</h2>
<h3>D1. 为什么散热层在叙事上必须存在（这决定“黑海域”的质感）</h3>
<p>只要你有大规模工业与配光网，你就会产生海量废热。废热最终只能靠辐射散出去。</p>
<p>辐射散热通量：
[
q=\sigma_\mathrm{SB}T^4
]
典型温度下：</p>
<ul>
<li>300 K：约 459 W/m²</li>
<li>350 K：约 851 W/m²</li>
<li>400 K：约 1452 W/m²</li>
</ul>
<p>这意味着：如果文明总耗散功率达到 (10^{16}\ \mathrm{W})（一万倍人类现今文明），哪怕散热片工作在 350 K，也需要面积：
[
A_\mathrm{rad}\approx \frac{10^{16}}{851}\approx 1.17\times 10^{13}\ \mathrm{m^2}
\approx 1170\ \text{万 km}^2
]
这是一片“大陆级”的散热场——<strong>它在可见光下很暗，但在红外里像海一样发亮</strong>。
所以你写“黑海域”会非常真实：肉眼里它黑，仪器里它亮。</p>
<h3>D2. 完好期：船坞带的功能（它是“飞船的机库”）</h3>
<ul>
<li>深空船坞：拼装、拆解、修复大型结构</li>
<li>质量驱动器物流：把资源（Layer E）发射成“货流弹道”，在此捕获</li>
<li>散热场：吸收整个系统的废热</li>
<li>应急隔离：发生碎片灾难时，船坞带可切断与主栖居带的交通，避免级联撞击</li>
</ul>
<h3>D3. 废弃后：黑海域如何形成（你要的“禁忌之海”）</h3>
<p>当控制失效：</p>
<ul>
<li>散热片失去姿态控制，很多会“折叠/翻转/乱飘”</li>
<li>船坞遗留大量断裂桁架与未完成结构</li>
<li>微小碎片在轨道上形成稠密的危险带</li>
<li>这里缺少“配光”，因此肉眼极暗</li>
<li>但偶尔会有废热残存导致局部红外强亮（你可以写成“夜里海面有不祥的暗红潮”——其实是热辐射）</li>
</ul>
<h3>D4. 场景切片（完好期 vs 废弃期）</h3>
<p><strong>完好期：船坞夜班（焊光无声，像雷暴）</strong></p>
<blockquote>
<p>船坞没有大气，焊接电弧每一次闪烁都像无声闪电。新一段遮光阵正在装配，薄膜在张紧框架上像水面一样颤动。远处，散热场以肉眼不可见的方式吞吐废热；只有在调试镜头里，它像一片缓慢起伏的暗海。</p>
</blockquote>
<p><strong>废弃期：探险者第一次进入黑海域（“世界”突然没有天空）</strong></p>
<blockquote>
<p>穿过天壁后的通道尽头不是山谷，而是一片没有空气的黑。你的火把照不到任何边界，光在真空里显得薄而冷。远处漂着巨大的骨架，表面有直线、有编号、有黄黑相间的警示纹——那种纹路不属于任何自然岩层。你第一次意识到：所谓“天壁”，不是山，是墙；所谓“禁海”，不是海，是机库的坟场。</p>
</blockquote>
<hr>
<h2>Layer E：资源捕获与“墓地轨道”（Resource / Grave Orbits）</h2>
<h3>E1. 完好期：资源链是“世界能活下去”的根</h3>
<p>你要支撑大气、海洋、土壤、工业，就必须持续补给：</p>
<ul>
<li>冰：水、大气（氧氮也可由冰与矿物化学制取）</li>
<li>碳：生态与材料</li>
<li>金属：结构、散热、镜阵、船坞</li>
</ul>
<p>Layer E 的工程形态可以写成：</p>
<ul>
<li>彗星/小行星捕获站（低推力拖曳、或动量交换）</li>
<li>分解与精炼站（把资源切成标准货流）</li>
<li>货流发射（质量驱动器把货物按弹道射向 Layer D 捕获）</li>
<li>墓地轨道（把报废结构送到稳定轨道，避免撞击主栖居带）</li>
</ul>
<h3>E2. 废弃后：墓地轨道成为“天空里的神”</h3>
<p>当文明断航：</p>
<ul>
<li>墓地轨道上的巨大残骸仍会反射恒星光</li>
<li>在主栖居体的天空里，它们表现为“移动的星”“缓慢的神舟”</li>
<li>偶尔有碎片坠落，成为“天降金属”“发光石”</li>
</ul>
<p>这能自然地把“飞船世界”的遗迹与宗教传说绑定起来。</p>
<h3>E3. 场景切片（完好期 vs 废弃期）</h3>
<p><strong>完好期：资源拖船与货流弹道（像航运）</strong></p>
<blockquote>
<p>拖船把一块冰小行星推上规定轨道，切割臂把它分成标准货块。货块被质量驱动器射出，沿着计算好的弹道飞向外层船坞带。你在控制室看见的是一条条细线——那是货流的航迹，像文明的血管。</p>
</blockquote>
<p><strong>废弃期：天空里的“天舟列阵”</strong></p>
<blockquote>
<p>每到冬末，夜空会出现一串极亮的“舟星”，它们沿同一条路径缓慢滑过。祭司说那是祖先归来巡夜。没人知道那其实是墓地轨道上的报废桁架在特定几何下集体反光；它们每 19.4 天重复一次——这是“自然年”的影子。</p>
</blockquote>
